Procedure: Xbox: A retail bundle is missing an item

[Asset 4049349]

Who should use this article?

  • Xbox

What is this article about?

This article describes how to handle issues when the customer reports that an item (accessory/game/DLC) is missing from a recently purchased Xbox retail bundle.

  • For other missing item scenarios, see KB 2988925 Process: Microsoft Store: Physical products/software Returns/Exchange/Engagement.

Overview

Common items that are missing from retail bundles:

  • Power Supply Unit (PSU)
  • Hard Disk Drive (HDD)
  • Controller
  • Game discs
  • AV/Component/Composite cables
  • Pre-Paid Code (PPC) tokens

Procedure

Hardware:
For missing hardware items (accessories, consoles, cables) and physical game discs:

  • Inform the customer that Microsoft understands and empathizes with their situation, however, any items missing from retail bundles are the sole responsibility of the retailer that they purchased the bundle from.
    Current manufacturing systems can detect the difference in weight of less than a single piece of paper. This ensures the retailer receives a complete system.
  • Direct the customer to return the full bundle to the retailer for exchange.

Tokens/PPC:
We can replace missing tokens if they meet the below criteria. Codes that do not meet the criteria cannot be replaced and should be directed to the publisher or retailer for assistance.

  • Is a physical token card – we do not replace tokens printed on a receipt
  • Came inside of a Microsoft-boxed product

All token replacements are required to go through the following process:

  1. Ask the customer for proof of purchase. See KB 818928 Procedure: Xbox: Code redemption issues (prepaid, trial, in-game, and subscription) for valid proof of purchase guidelines.
  2. Check to see if there's a replacement code in the token tool.
  3. Issue a replacement code if there is one available. If there is no replacement code available, engage your Ranger or TA for assistance.

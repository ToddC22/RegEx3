Guidance on completing the template

Use:

    • This stand alone template is intended to be used internally. However, the same information can be provided as part of a customer facing email as necessary.
  • Use action plan template or summary template every day there is customer/partner contact to document next contact and next actions.

Contact Date Time:

  • Do not commit to a specific call back time, instead try to agree a call-back window (for example, Thursday or Thursday 9AM-1PM). This gives freedom to balance your workload and avoids customer/partner disappointment if you cannot make the agreed time.

Action on Customer/Partner:

  • If sending this action plan template to a customer and you have provided the customer with 3rd party information you must include the following disclaimer:

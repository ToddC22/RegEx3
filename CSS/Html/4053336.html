1. Services section overview

All available services are listed by default in this section. Major services are called out with their own icon, everything else will appear under Other Services. If the customer has an active service, then it will show a solid blue dot next to the service icon. If there is no blue dot, then the customer does not have that service. If there is a light blue circle, it means that the customer had a previously active service. Selecting a service icon will take you to a more detailed view of that service.

[Asset 4464479]

Each section will show the navigation breadcrumb, a list of subscriptions, and the action menu icon with any available actions. Common subscription management options are:

  • Change payment instrument: Move the subscription or charge to a new payment instrument.
  • Cancel subscription: Cancel a subscription immediately.

Note:

When canceling a subscription, turning off auto-renew, or removing payment instruments, you must perform a Save and Retention workflow. After selecting a cause for the action, talking points will be provided.

  • Show all orders related to subscriptions (In this view, only top five orders will be shown. Advocates can select View all for this Subscription to see all orders.) Turn Recurring billing on/off: Manage the recurring billing status of a subscription.
  • Convert a subscription.

2.2 Deactivate device

Follow instructions found in KB 2916962 Procedure: Microsoft 365 save and retention when you start to process canceling a subscription, turning off auto-renew or converting a subscription. Once you select a cause for any of the above actions, prompts will appear on the screen to help with the Save and Retention process.

For PC and Mac, the Office device can be deactivated by selecting the action menu icon next to the device. This is not available for tablets or mobile devices, so no action button will be visible for those devices.

3.2 Trusted Console

Use the steps in the following table to complete Trusted Console password reset.

Perform Trusted Console:

  1. Select on Xbox Live in the Services section of MST.
  2. Select the action menu icon to the right of the Gold Member avatar.
  3. Select Trusted Console.
  4. Review the information on the screen, then select Reset ‘Required Password’ settings to continue.
  5. Review the updated statement at the top and select Reset ‘Required Password’ settings to continue.

3.4 Xbox Live > Change Payment Instruments

Change Payment Instrument is used to select another payment method already associated with the account. Follow the steps below to change a payment instrument:

[Asset 4464624]

  1. Select the action menu icon and select Change Payment Instrument.
  2. Select a new payment instrument from the New PI dropdown list.

[Asset 4464625]

Enter comments and select Submit to save the change.

Note: If the payment instrument is not listed, you first need to add a payment instrument by using the action menu icon  located at the right side of the customer’s detail (top of the MST page) and select Add PI.

[Asset 4464626]

  1. In MST, go to the Services – Xbox Live – Manage Subscriptions section. Search for the subscription on which you need to turn on/off auto-renew.

[Asset 4464629]

  1. Select the action menu icon of the Xbox subscription. Select Turn on Auto Renew. The menu will update itself based on the current state of Auto Renew for that subscription.

[Asset 4464630]

  1. Confirm the action and payment instrument with the customer. Add text in the Comments field and select Next.
  2. Review the change and select Submit.
  3. The state of Auto Renew has now been turned On. Confirmation shows the date the subscription will renew, the price and the payment instrument. Select Ok.

6. Family

Selecting the Family icon takes you to the Services > Family page. This shows Family information pertaining to adults, children, and pending members.

From here, you can remove family members or manage a child account settings. Family settings can only be managed when a parent’s profile is loaded and authenticated.

Note: The option to add a member is greyed out and is not actionable for security purposes. To add a family member, customers must navigate to family.microsoft.com and follow the steps located here.

6.2 Manage children settings

There are 2 settings that can be changed/updated from MST on behalf of the customer:

  • Turn on/off the adult approval for child’s purchase (Ask to Buy).
  • Change/update the Maturity Level for Apps, Movie, Game, TV, Music.

To update the children settings, follow the steps below:

  1. Load and authenticate a parent’s account within the family.
  2. Select the child in the family and select the action menu icon.
  3. Select Manage Settings. [Asset 4464831]
  4. Select the appropriate settings. [Asset 4464833]

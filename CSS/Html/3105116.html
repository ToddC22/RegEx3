What is this article about?

This article provides information about Windows 10 activation, and how to troubleshoot activation errors.

Note: This process does not apply to SMB or Commercial customers

Related public content about activation errors:
 
Activation in Windows 10
Get help with Windows 10 activation errors

 

Important first steps

 

When you are troubleshooting Windows 10 activation errors, the customer should be on the latest version of Windows 10, including updates. From Start, search for check for updates, select Check for Updates from the search results, and then follow the on-screen instructions.

In version 1709 or higher, it is important to note that the PID can’t be used to find the product key. It will show the edition, but if you put the PID into the KIT tool, it will show as blocked because the PID matches a default key, even if the customer used a valid product key to install. 

Also: Original Equipment Manufacturers (OEMs) cannot hand out product keys or provide activation overrides. The OEM is responsible for providing new PCs with a licensed Operating System at the point of purchase. Once activated for the first time, any action that causes a system to become non-activated is the responsibility of Microsoft Support to help address with the user.

1. The Activation troubleshooter in Windows 10 version 1607 and later





On a non-Activated PC with Windows 10 Anniversary Update or later, select Settings, Update and Security, and then Activation. You will see the Troubleshoot button. The troubleshooter can help resolve several known issues a few of which are listed below. Please run the troubleshooter as the first step to any activation error.

If the customer is signing in with a local account, they might be prompted to change to a Microsoft Account (MSA) sign in. This will allow the troubleshooter to check if the customer's license is already associated with their MSA, or allow the customer to associate it now. Only MSA sign ins can be associated with the license, local accounts can’t.
 
  1. If the user has a digital license associated to their PC for a different edition, the troubleshooter can detect this and suggest a solution to the customer. If they have installed Windows 10 Home and have a license for Windows 10 Pro the troubleshooter will offer to upgrade the edition to match. If they have installed Windows 10 Pro and they have a license to Windows 10 Home, they must backup their data and do a clean install of Windows 10 Home Edition.
  2. If the customer has changed their motherboard, their system might need to re-activate.

1.1 Re-Activation of a Windows 10 digital license after a user has made a motherboard change (version 1607 or later only)


The Windows 10 digital license is tied to the HardwareID of the PC. If the user changes their motherboard, this HardwareID changes, and the PC will become non-Activated. In some cases, the user can solve this by running the troubleshooter and re-activating their PC with their existing digital license. For this to happen, they must:
  1. Be connected to the internet
  2. Have been running Windows 10 Anniversary Update (version 1607) prior to changing their motherboard
  3. Signing into Windows 10 using a Microsoft account

To launch the troubleshooter, select Settings, Security and Update, and Activation. You will see a Troubleshoot button on non-activated PCs. After running this, if the user is still not activated they are offered the link “I changed hardware on this device recently. If the user is not already running a Microsoft Account, they will be asked to sign in to see a list of eligible PCs associated with the users Microsoft account. The user can select the correct one and have their PC re-activated.





If the user does not have an eligible PC or the re-activation fails, see 3082672 Policy: MPA: Windows 10 Motherboard Replacement (MBR) and Device Exchange

2. General information about Windows 10 Activation


Activation helps verify that your copy of Windows is genuine and hasn’t been used on more devices than the Microsoft Software License Terms allow. Depending on how the customer got their copy of Windows 10, activation will use either a digital license (formerly called digital entitlement) or a 25-character product key.
 
How the customer got Windows 10 Activation method
Upgraded to Windows 10 for free from an eligible device running a genuine copy of Windows 7 or Windows 8.1, prior to July 30, 2016.

Or

Windows Insider upgraded to the newest Windows 10 Insider Preview build on an eligible device that was running an activated previous version of Windows and Windows 10 Preview.		 Digital license
Purchased from the Microsoft Store and successfully activated Windows 10 one of the following products:

· Genuine Windows 10

· Windows 10 Pro upgrade		 Digital license
Purchased a copy of Windows 10 from an authorized retailer. Product key

(On a label inside the box Windows 10 came in.)
Purchased a digital copy (ESD) of Windows 10 from an authorized retailer. Product key

(In the confirmation email you received after buying Windows 10 or in a digital locker accessible through the retailer’s website.)
A Volume Licensing agreement for Windows 10 or MSDN subscription. Product key
Purchased a new device running Windows 10. Product key

(Pre–installed on your device, included with the device packaging, or included as a card or on the Certificate of Authenticity (COA) attached to the device.)


Digital license is made available to a device which upgrades from Windows 7 or Windows 8.x to Windows 10. Starting Windows 10 Version 1511 (November update), the customer will be able to see information on the Activation page when a device is activated using a digital license.



When you upgrade from Windows 7 or 8.1 to Windows 10 or if the PC is activated using a digital entitlement, then no Product Key is entered and a 5x5 Default Key remains with an associated default PID. This 5x5 key is a generic key and cannot be used for activation of Windows 10. When a generic key or the IID of such a key is looked up in PACSR or KIT, it will show as blocked.

5.4 Error Code 0x87E10BC6

An error occurred with our activation server or licensing service. Wait a few minutes and then try the following Windows 10 Pro License Recovery steps.

Select the Start button, then select Settings > Update & security > Activation , and then select Troubleshoot to run the Activation troubleshooter.

When the troubleshooter is complete, select Go to Store to launch the Store app. If the Store app shows there was a problem, select Try again.

After a few minutes, you should see a message thanking you for activating genuine Windows, followed by another message showing that Windows was activated.

Windows reported that the hardware of your device has changed. Error Code 0xC004F211

N/A

You changed the hardware on a previously activated Windows 10 PC.  A Windows license is associated with your PC hardware. If you make significant hardware changes, such as replacing your motherboard, Windows won’t be able to find a matching license the next time you start your PC.  If Windows was preinstalled on the PC, then a new license is required if you have made a change such as replacing the motherboard. To purchase a new Windows license, select the Start button > Settings  > Update & Security  > Activation , and then select Go to Microsoft Store.  

If you have a Windows product key you used on your PC before the hardware change, enter it by selecting the Start button > Settings  > Update & Security  > Activation  and then selecting Change product key. 

In addition, Microsoft provides an exception path for transferring a Windows License after a hardware change. You’ll need to be signed in to the same Microsoft account before making the hardware change and to use the exception path.  

To access this exception path, the Start button > Settings  > Update & Security  > Activation  and then select Troubleshoot. If the troubleshooter can’t resolve the issue, you’ll see a link to reactivate after a hardware change. Select it to follow this path. For more info, see Reactivating Windows 10 after a hardware change. 

 

Agent: If the above steps don’t resolve the issue, and the customer has a 5x5 product key, perform an override.

The product key used on this PC didn't work with this edition of Windows. You might need to get in touch with the store or company you bought Windows from, enter a different product key, or go to the Store to buy genuine Windows. Error Code 0xC004F212

N/A

You reinstalled the wrong edition of Windows 10.  When you reinstall Windows 10 using a digital license, you’re asked to choose which edition to install. If you choose an edition other than the one you’re licensed to use, you may receive the above error.  

If you think you installed the wrong edition, select the Start button > Settings  > Update & Security  > Activation  and then select Troubleshoot. If the troubleshooter detects a valid Windows 10 digital license for an edition not current installed, it will let you know, and show you how to install the correct edition.

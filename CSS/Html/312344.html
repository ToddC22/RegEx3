Request for Certificate Is Denied and a "The Request Subject Name Is Invalid or Too Long" Error Message Occurs

If you enroll for a certificate, the request may be denied and you may receive the following error message:
The request subject name is invalid or too long. 0x80094001
In addition, the following message may be logged in the event log:
Certificate Services denied request number because the request subject name is invalid or too long. 0x80094991...
This issue can occur if the subject in the Name box matches the name of the certification authority (CA).
The subject's CN is string-compared and binary-compared to the issuer's name. If the full DN names are available, these names are compared before a certificate is issued. Comparison does not depend on the actual encoding of the characters in the subject's name.

2. Definitions

  • Country code: This is a short alphabetic or numeric geocode created to represent a country and/or dependent areas. For Call Extensions, this is where the phone number is registered. The country code must contain a two characters country code. The supported country code values include AU, GB, ID, IE, IN, MY, NZ, PH, SG, TH, US, and VN.
  • Call only: This option will determine whether the phone number is the only clickable item in the ad.
  • Only my phone number: The phone number is the only link that a user can click.
  • Both my website and phone number: This will show both the website and the phone number.
  • Call tracking enable: This option helps advertiser tracks call details such as call start/end time, missed/completed calls, call duration and call type.
  • On all devices using a Microsoft Advertising forwarding number: Microsoft Advertising no longer offer the option to select a toll free/local number. All advertisers will default to using a local area-code number when they opt into using Microsoft Advertising forwarding number.

US and UK customers who had previously set up Call Extensions using a toll-free forwarding number will be allowed to keep them (with no action required) until the extension(s) are deleted or the forwarding number otherwise expires.

  • On smartphones using my phone number: This will show the phone number provided.

Note: No phone number will show on a Desktop or tablet by enabling this feature.

3. Best practices and policies

Call Extensions allow advertisers to display a phone number directly in their search ads, enabling customers to click to call or dial that number.

Call Extensions help the advertiser’s ads stand out on the page, giving customers a convenient way to call their business directly from search results — one click to call is all it takes. More calls mean more leads and faster purchasing decisions.

For many local businesses, phone calls continue to generate the best leads. Phone calls are typically 10 times more likely to result in a sale versus a click. With Call Extensions, your phone number is placed alongside on the webpage in search results. Customers can easily find and call the number on smartphones, desktops and tablets. The advertisers are only charged for actual call. They should include these links in their ads to drive more calls for their business.

[Asset 4503999]

Advertisers with Call Extensions generated 3-6 percent more clicks, on average, than those without Call Extensions. Clicking on the phone number link will initiate a call to that number and advertiser will be charged up to their bid amount (just as if the user had clicked on a website link).

Call Extensions for all devices are available in the U.S. and UK only. Click-to-call ads on smartphones are available in all markets.

3.1 Product details

  • There are two types of Call Extensions and Reporting:

All devices (PC, Tablet, Smartphone) – US and UK only

  • This is a Bing-provided forwarding number that displays with your ads.
  • More enhanced reporting is available that includes metrics such as call type, duration of the call, and area code.
  • When making a call on PCs or Tablets, customers reach advertiser via a Skype Call.
  • When making a call on smartphones, customers can click to call by tapping on the phone number in the ad.

Smartphone only – Global markets (except China)

  • Advertiser own phone number will be displayed.
  • Regular click reporting is available.
  • When making a call on smartphones, customers can click to call by tapping on your phone number in the ad.
  • Call Extensions will display in mainline.
  • Advertisers are charged as a standard cost per click for each call.
  • Call Extensions will serve alongside with other ad extensions.
  • Call Extensions can be uploaded in Microsoft Advertising UI and in Microsoft Advertising Editor.
  • Call Extensions can be imported from Google Ads into Microsoft Advertising Ads.
  • Call Extensions can only be uploaded at the Campaign level.
  • Advertisers can have one Call Extension associated per Campaign.
  • Can track performance of Call Extensions by running the following reports:
  • Ad Extension by Keyword
  • Ad Extension by Ad
  • Call Details reports
  • Call Extensions have the added benefit of allowing advertiser to designate their ads as call only ads. When they select call-only ads, website links in the ads are not clickable from a smartphone. Instead, the phone number is the only link that a user can click. Clicking on the phone number link will initiate a call to that number and advertiser will be charged up to the bid amount (just as if the user had clicked on a website link).
  • Call Extensions can be used together with Location Extensions. When these two features are used at the same time, the phone numbers displayed, and the availability of click-to-call, depend on the device where the ad is being viewed.
  • Advertiser can use the campaign-level day and time targeting options to have Call Extensions appear only when they want them to.
  • If a forwarding number is used, advertiser can access reports that provide details like call duration and missed calls on all calls the ads have generated.

3.2 Policies

  • To get a forwarding number, an ad group within a campaign using Call Extensions must have at least 5 clicks and $5.00 in spend for the previous month's billing cycle.
  • Advertiser cannot target specific locations with Call Extensions.
  • Valid phone number: The phone number must be in the correct format which includes a valid area code. For example, the correct format for the United States is: (425) 555-0177.
  • International calls are not allowed.
  • The phone number must be all numbers.
  • No letters are permitted.
  • The phone number can contain a maximum of 35 characters and must be valid for the specified country.
  • Phone number correction (AES Policy): If a phone number has an invalid phone number format, make sure requestor will be informed.
  • Phone Number Formatting: Microsoft Advertising will automatically format the call extension with hyphens.
  • Disallowed characters in phone number: The phone number cannot contain special characters except for parentheses and a dash.
  • Fees: Phone charges more than standard fees must be clearly identified before the call is placed, including disclaimers in ad copy. At times, Microsoft Advertising reserve the right to block certain premium rate phone numbers (such as "900" numbers in the U.S.).
  • Call recording: In addition to complying with all applicable legal requirements, a clear notification of the fact that a phone communication is being recorded is required before recording begins.
  • Call forwarding: For Microsoft Advertising to provide a forwarding number, an advertiser ad group must have at least 5 clicks and $5 in spend for last month’s billing cycle and a rolling window of 30 days.
  • If the campaign includes call and location ad extensions, this phone number will override the phone number specified in the location ad extensions.

US and UK customers who had previously set up Call Extensions using a toll-free forwarding number will be allowed to keep them (with no action required) until the extension(s) are deleted or the forwarding number otherwise expires.

4. ASD coding

Ticket status (choose the one that best fits your case): Completed, pending, or cancelled 

Customer issue: Optimization tactic request

Topic: Extensions

Scenario: Call Extensions creation or update

Problem: Call Extensions creation or update

Solution delivered (choose the one that best fits your case): Call Extensions Created or Updated, Call Extensions Request Cancelled, Solution in Progress.

Requirements: Enabling (Uploading and Associating) Call Extension requests are coded as Call Extensions Creation or Update. These requests are for net-new Call Extension.

The BOS advocate is responsible to create a new, separate ticket in the following scenarios:

  • Delete existing Call Extension then, enable new Call Extension. The BOS advocate is responsible to create a new, separate ticket for deleting an existing call extension (total of two separate tickets for the request). The delete portion of the work follows the requirements in KB 4039244 AES: Process: Microsoft Advertising: Uploads (Bulk Campaign Management).
  • Edit or delete existing Call Extensions. This request is coded as Uploads and follows the requirements in KB 4039244.

The BOS advocate is responsible to provide the new case number for each portion of the work to the requestor in the original request to set proper expectation. Then, the newly created ticket/s must have a separate email communication and proper documentation in the ASD Case Notes must be completed.

5. Enabling, Editing, and Deleting Call Extensions via Microsoft Advertising UI

5.1 Uploading Call Extensions using import file

The following section provides instructions on how to upload call extensions via Microsoft Advertising UI.

  1. Before uploading call extensions, you must first Gather Requirements. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.
  2. In Microsoft Advertising UI, select Import Campaigns in the top-level tabs.

[Asset 4504000]

  1. Then select Import from a file.

[Asset 4504001]

  1. Select the import file for Call Extension. Then select Continue.

Note: You can download the import template for the required headers available in the UI.

[Asset 4504002]

Below are the headers when uploading a Call Extensions:

Type Campaign Phone number Country code Call only Call tracking enabled
Campaign call Ad Extensions Campaign 1 (661)298-2000 US No Yes
  1. Once the file is imported, review the column matching, then select Continue.
  2. Choose Import Options by unticking all the checkbox and select the checkbox for Import new items, Ad extensions, and Call extensions only. Then select Import.

[Asset 4504003]

  1. The Import preview will show up on the page. Then select Apply changes.
  2. The Import Summary will show on the page. The number of newly added Call Extension will show in Ad Extension row, under New column. Then select View imported campaigns.

[Asset 4504004]

Note: In case that there are skipped items, there will be a link where you can download all the skipped items.

  1. Once done, you can check the uploaded Call Extensions under Ad Extensions tab.
  2. Verify your request has been 100% completed. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.

5.2 Associating Call Extensions

The following section provides instructions on how to create call extensions on Microsoft Advertising UI.

  1. Before associating call extensions, you must first Gather Requirements. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.
  2. Select Ad Extensions in the top tab area.
  3. Select Call Extensions from the drop menu.
  4. Select Create Ad extension.
  5. Select the Campaign you’d like to associate with this call extensions.

[Asset 4504006]

  1. Select Add new Call Extension.
  2. A new window will show up and fill in the details provided by the requestor. Then select Save.

[Asset 4504007]

  1. Repeat above steps for all requested campaigns.
  2. Send an email to requestor using standard template to notify them that the request is completed.
  3. Verify your request has been 100% completed. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.

5.3 Editing Call Extensions

The following section provides instructions on how to edit call extensions on both Microsoft Advertising UI and Microsoft Advertising Editor.

  1. Before editing call extensions, you must first Gather Requirements. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.
  2. Select the Ad Extensions tab.
  3. Select Call Extensions from the drop-down menu.
  4. Select the checkbox of the Call Extension that you would like to edit.
  5. Select Edit.
  6. Then select Edit an extension.
  7. A pop window will show to modify the existing Call Extension details. Change the settings as per submitter’s preference.
  8. Then select Save once the Call Extension has been edited.

[Asset 4504008]

Note: Any edits to a call extension also impact all campaigns associated to it.

  1. Once done, perform a quality check if the changes were made on the call extension.
  2. Verify your request has been 100% completed. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.

5.4 Deleting Call Extensions

The following section provides instructions on how to delete call extensions in Microsoft Advertising UI.

  1. Before deleting call extensions, you must first Gather Requirements. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.
  2. Select the Ad Extensions tab.

  1. Once done, perform a quality check if the changes were made on the call extension.

6.2 Uploading Call Extensions via Import from a File

  1. Before uploading call extensions, you must first Gather Requirements. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.
  2. Open and review the import file provided by the requestor. Make sure you have all needed information. If no import file was provided, create an import file with the following columns:

Type

Campaign

Phone number

Country code

Call only

Call tracking enabled
  • Type: Input Campaign Call Ad Extension
  • Campaign: Campaign name should be specified in the request.
  • Phone number: Must be in a correct format with area code.
  • Example: The correct format for the United States is (425) 555-0177.
  • Country Code: The country code where the phone number is registered. The country code must contain a two characters country code.
  • Example: The correct country code for the United States is US and for United Kingdom is UK.
  • Call Only: This will determine whether the phone number is the only clickable item in the ad.
  • Only my phone number (Yes): The phone number is the only link that a user can click.
  • Both my website and phone number (No): This will show both the website and the phone number.
  • Call Tracking Enable: To enable call tracking, set this element to Yes. To disable call tracking, set this element to No. If not specified, the default value is No.
  • On all devices using a Microsoft Advertising forwarding number: Microsoft Advertising UI no longer offer the option to select a toll free/local number. All advertisers will default to using a local area-code number when they opt into using Microsoft Advertising forwarding number.

6.6 Deleting Call Extensions

  1. Before deleting call extensions, you must first Gather Requirements. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.
  2. In Editor, select Ads and extensions. Then select Call Extensions under Manager pane.
  3. Select the existing Call Extension that you’d like to delete to highlight.
  4. Select the Delete button and an X will indicate that the Call Extension is deleted.
  5. Select Post to upload the changes made.
  6. Verify your request has been 100% completed. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.

If the request is to delete Call Extensions in Bulk, there is an option to Delete multiple Call Extensions:

  1. Before deleting call extensions in bulk, you must first Gather Requirements. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.
  2. In Microsoft Advertising Editor, select Ads and extensions. Then select Call Extensions under Manager pane.
  3. Select Make multiple changes.
  4. Then select Delete multiple Call Extensions.
  5. Select Insert Headings to make it clear what fields need to be copied and pasted. Copy and Paste the Call Extensions Data into Microsoft Advertising Editor.
  6. When you’re done copying and pasting the call extension information, select Next.
  7. Once this has been submitted, you’ll see the summary view that shows you have deleted call extensions and the number of the deleted items.

[Asset 4504022]

  1. Post the changes.
  2. Verify your request has been 100% completed. To do this, see KB 4493123 AES: Process: Microsoft Advertising: Gathering Requirements for a change request and submit for Requestor Review.

7. Additional tips

Another way of Enabling, Editing, and Deleting Call Extension is through Shared Library.

  1. Under Call Extensions in Microsoft Advertising Editor, select the existing Call Extension to highlight.
  2. In the Editor pane, click the blue link Edit these call extensions in Shared Library.

[Asset 4504023]

  1. The Shared Library will show up with the list of existing Call Extensions. Highlight the Call Extension that you’d like to edit, then make the changes in the Editor pane.

[Asset 4504024]

  1. Select Post to upload the changes.
  2. Send email to requestor using standard template to notify them work is completed.

[Asset 4504025]

Note: Adding and Deleting can also be done in Shared Library by navigating on the Microsoft Advertising Editor tools.

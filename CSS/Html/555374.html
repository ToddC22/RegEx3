Installation of Windows Server 2003 SP1 fails with "Error 0x80092026: The cryptographic operation failed due to a local security option setting."
1. Open regedt32 and browse to HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\SystemCertificates\TrustedPublisher
2. Under TrustedPublisher look for Safer key and delete it.
3. Begin the Installation of Windows Server 2003 SP1.

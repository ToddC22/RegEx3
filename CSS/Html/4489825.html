Procedure: VLS: EMEA Escalation using Service Desk

[Asset 4489629]

Who should use this article? 
EMEA Tier 1 VL agents.

What is this article about?
This process describes how to proceed with escalating a case to Tier 2 using collaboration tasks and the escalation queues in Service Desk. It should be followed when the customer’s request cannot be handled directly by Tier 1 and need to be escalated to a specialized escalation agent that could resolve it or escalate it further.

Whenever a case needs to be escalated to Tier 2, you need to use the corresponding template that you will find in each group, under the ‘VL Queues’ section, fill it up completely and post it in the description whilst you transfer the case or in the DTM for specific escalation subjects. More on this under ‘VL Queues’.

In Agent Manager, a Team Manager needs to assign the queue in question to the Tier 2 agent. In this document, the queue “EMEA_VL_OSA Key Replacements” will be used as example:

Edit the case, click the [Asset 4489850] icon and select “Assign to Me” to start working on the task and set it to “In-progress” (At this point, the task will disappear from the “Unassigned” queue and the collaboration icon will disappear for the frontline advocate):

Once the work on the task has been completed or if you need more info, create a follow up tasks for the agent.

If the agent has an existing follow up task already change it edit to your own follow up:

[Asset 4489862]

If the agent does not have a follow up tasks yet, you can create a new one for the agent.

Click on + Follow-up to create new Follow-up for the agent and search for the agent name.

[Asset 4489863]

Add notes to the description:

  • Escalation – clarification/ more details needed, please see case notes
  • Escalation – case is solved, please see case notes

Select a time within two business hours (you only see your own time in the menu). For example, if two hours from now is outside of the Hours of Operations (HOOPS), then choose the corresponding time for the beginning of the next business day.

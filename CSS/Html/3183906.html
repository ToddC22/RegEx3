Procedure: Windows: How to install an earlier Windows 10 version if the customer can't install the most recent version

[Asset 4049349]

Who should use this article?

  • Windows

What is this article about?

This article provides troubleshooting steps to help customers install earlier versions of Windows 10 when they are unable to install the most recent version.

Overview

If installation of the most recent Windows 10 feature update fails, or the customer insists on going back to an earlier version and uninstall is not available, provide the customer with the Media Creation Tool (MCT) location for the two previous versions.

Download links for Windows 10 November 2019 Update (version 1909)

  • Executable: https://go.microsoft.com/fwlink/?linkid=872132
  • Catalog: https://go.microsoft.com/fwlink/?linkid=872133

Download links for Windows 10 October 2018 Update (version 1809)

  • Executable: https://go.microsoft.com/fwlink/?linkid=823664
  • Catalog: https://go.microsoft.com/fwlink/?linkid=860101
  1. Download the executable version of MCT executable for the appropriate version from the link above, and save it locally (for example, C:\Users\<username>\Download\).
  2. Download the version of MCT catalog for the appropriate Windows version from the link above and save it to the same location as above in step one.
  3. Rename the catalog file downloaded from step #2 from "products_xxxxxxxx.cab" to "products.cab".
    • Important: Ensure that show File name extensions is enabled before taking this step.
    • This can be enabled by opening File explorer -> selecting View -> checking the box next to "File name extensions" on the Show/hide tab.
  4. Open an admin Command Prompt window.
  5. Launch the MCT executable with the "/selfhost" switch (e.g. "C:\Users\<username>\Downloads\MediaCreationToolxxxx.exe /selfhost").
    • Replace xxxx with the version number of the executable downloaded in step 1.
  6. Assist the customer with the MCT download and installing the earlier version of Windows 10.

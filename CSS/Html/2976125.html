Outlook Web App (OWA) escalation procedure for Office 365

Known Outlook Web App (OWA) issues that were escalated previously to the Exchange PG where the issue was determined to be solvable through configuration changes without needing escalation.

Prior to escalating to the Exchange PG ensure that the issue cannot be solved by the suggested troubleshooting actions below:
On Logon to OWA or other actions in OWA, user sees an error that contains this informationAction to take
Microsoft.Exchange.Data.Storage.InvalidLicenseException

1)      Check that the license is correctly assigned and not recently expired

2)      If issue persists, troubleshoot as a Provisioning issue

1)      Check the OWA mailbox policy of the mailbox and make sure that both of these properties are not set to false:

·         OWALightEnabled : False

·         PremiumClientEnabled: False

OWA

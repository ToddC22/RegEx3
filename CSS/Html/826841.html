256986 Description of the Microsoft Windows registry
Microsoft Systems Management Server (SMS) 2.0 and SMS 2003 clients can use the Remote Control icon (Smsrc.cpl) in Control Panel to change the client-side remote control settings. However, sometimes you may want distribute a silent package that automatically changes a registry setting. To do this, advertise a package to modify the client's registry.
When you use the Remote Control tool in Control Panel to change the remote control settings, the Remote Control Agent is notified, and the changes occur immediately. However, if you modify the registry to make the changes, the changes may not take effect until the SMS client computer restarts.

The remote control registry values are located under the following subkey:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\SMS\Client\Client Components\Remote Control
Warning Serious problems might occur if you modify the registry incorrectly by using Registry Editor or by using another method. These problems might require that you reinstall your operating system. Microsoft cannot guarantee that these problems can be solved. Modify the registry at your own risk.

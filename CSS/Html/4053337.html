5. Error codes encountered in payment instruments

You may encounter non-descriptive error codes when assisting customers via Purchase on Behalf of (POBO) in MST. These error codes are not specific to POBO or MST but same error codes when customer encountering the error during purchase through Microsoft Store website. The following list doesn’t contain all possible error codes but does account for the majority of occurrences.

If an error code is encountered that is not listed below, please escalate to Tier 2 for submission of a TPI with screenshots and URLs so this list can be updated in the future.

Sample error code :

8. Transfer balance (all LOBs)

Use the following steps to transfer a balance from one PI to another PI from a customer’s account. This feature is used to transfer a Ready Balance from one PI to another.

Note: Transfer Balance only shows up as an Action if the PI has a Ready Balance.

  1. Select the PI you want to transfer FROM and select Transfer Balance.
  2. Under Transfer Ready Balance To, use the drop-down, select the customer’s PI that you want to transfer TO.
  3. Enter comments detailing why the transfer is being made and select Next.

1. Quick Assist overview

Quick Assist is a Windows 10 app that provides remote assistance access to customer PCs to resolve system related issues.

Microsoft Quick Assist standard app

The standard version of Quick Assist is available to all Windows 10 users and has all common Remote Desktop sharing features such as:

  • Control mouse
  • Control keyboard
  • Select monitor
  • Annotate
  • Actual size/fit to screen
  • View mode (take full control or view screen only). This feature is only available in the Windows 10 (1809) - October 2018 Update/RS5.
  • Restart
  • Task Manager
  • Pause
  • End

Microsoft Quick Assist for Support app

Microsoft Quick Assist for Support is only available to CSS advocates. Consumer Support advocates/ambassadors will sign in using their Service Desk UPN credentials. This will allow customers to see that the person supporting them is a representative from Microsoft.

In addition to the features in the Standard version of Quick Assist the Support version also contains:

  • Instruction channel
  • Safe Mode restart (only available when signed in with a UPN account and the customer on a wired LAN connection)
  • Mid-session reconnects
  • Restart and reconnect
  • Session cancelled by Helper
  • Session cancelled by Sharer

Note:

  1. To experience all the features in the Microsoft Quick Assist for Support app, users should sign in with their AAD credentials.
  2. Use the Instruction channel to interact with the Sharer during the session but only to exchange links, commands, error codes, and other instructions. This should not be used as a substitute chat channel.

Microsoft Quick Assist can be with customers who have Windows 10 RS 1 and later installed on their computers.

3. Reconnecting from a disconnect

Step

Image
  1. If the session was disconnected for various reasons (third-party app/service restarts the system/crash), the customer should launch Quick Assist again. Quick Assist will attempt to reconnect the customer back to the previous session.

[Asset 4496432]
  1. If the timer for reconnecting runs out (approximately five to seven minutes), this message will be displayed on both your and the customer’s device. In this scenario, you and the customer should both relaunch Quick Assist and start a new session.

[Asset 4496434]
  1. If the timer for reconnecting runs out (approximately five to seven minutes) and you have closed Quick Assist or do not have it running, the customer will get this message. In this scenario, you and the customer should both relaunch Quick Assist and start a new session.

[Asset 4496435]
  1. If the timer for reconnecting runs out (approximately five to seven minutes) and the customer closes Quick Assist or does not have it running, you will get this message. In this scenario, you and customer should both relaunch Quick Assist and start a new session.

4. FAQ

Q: What is Microsoft Quick Assist and are there different versions?

A: Microsoft Quick Assist is a Windows 10 app that allows Microsoft Support Advocates to remotely access a customer computers to solve system related issue faster or educate customers to help themselves solve their issue.

Microsoft Quick Assist works between two people:

  • Sharer: The person who needs support and shares their computer
  • Helper: The Microsoft Support Advocate who provides assistance.

The Standard version of Quick Assist is available to all Windows 10 Users. It has all common Remote Desktop sharing features such as:

  • Control Mouse
  • Control Keyboard
  • Select Monitor
  • Annotate
  • Actual Size/Fit to Screen
  • Restart
  • Task Manager
  • Pause
  • End

Microsoft Quick Assist for Support is only available to CSS Advocates. In addition to the features in the Standard version of Quick Assist the Support version also contains:

  • Instruction Channel (allowing helper and sharer to exchange links, commands, error codes and other instructions).
  • View Mode (Take full control or View screen only).
  • Safe Mode restart (only available when signed in with a UPN account and the customer on a wired LAN connection).
  • Mid-session reconnects.
  • Restart and reconnect.
  • Session cancelled by Helper.
  • Session cancelled by Sharer.

Q: What is some interesting information about Microsoft Quick Assist?

A:  

  • Users are prompted to accept the Microsoft Support Agreement when the Helper is a CSS advocate.
  • Downloadable Win32 package of Quick Assist.
  • Helpers can hear audio output from the Sharer’s device.
  • Events are logged locally within Event Viewer.
  • Keyboard input is mapped to the Helper’s keyboard layout.
  • The Helper can enter Full Screen mode on their device.
  • Sessions are terminated after 30 minutes inactivity and no response from users.
  • The Clipboard Sharing of text from Helper to Sharer.
  • The Instruction Channel automatically opens when text is received.
  • The Helper can transfer files to and from the Sharer device.
  • Quick Assist launches with appropriate permissions.
  • Unsupported platforms:
    • Windows XP (EOS 4/18/2014)
    • Windows 7 (EOS 1/14/2020)
    • Other Windows platform support (server, etc.) will depend on availability of compatible APIs for remote support. We are still trying to determine what API gaps exist.

Q: Does Microsoft Quick Assist have collaborative sharing?

A: Not at this time, but it is a feature being worked on.

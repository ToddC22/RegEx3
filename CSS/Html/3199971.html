Who should use this article?

What is this article about?

This article will help you address common performance and maintenance issues that your customer may encounter.

1. Windows - hangs or freezes

Confirming the issue

When the Surface is on and being used:

  • Windows will stop responding and freeze on the screen.
  • No errors may be displayed.
  • The Surface is not responsive to input from the touch screen, trackpad, or keyboard.

Confirmation/paraphrase

"Just to make sure I understand your issue… While you were using your Surface, it stopped responding and is no longer responsive and the screen appears frozen.”

How to code

After confirming the issue, code the case.

  • Windows OS > Performance and Maintenance > Performance - Windows > Windows – hangs or freezes

Resolution steps

[Asset 4486344]

Step

How

Why

1. Copy and paste the template to the right to begin your note taking for this scenario. Document your findings for every step performed.

Notes template:

  • Check known issues >>
  • Force the device to shut down and restart >>
  • Run SDT or Windows Updates >>
  • Check Reliability Monitor >>
  • Scan for Virus, spyware, or other malware >>
  • Check Task Manager >>
  • Perform a Memory diagnostic >>
  • Reset >>

This ensures a consistent format for capturing and reviewing case information. Performing this standard procedure helps prevent instances where a step was neglected to be performed.

2. Check known issues.

From KB 4072657 Informational: Surface: Technical guides and known issues landing page, select the customer’s Surface model and check for the issue in the “Windows OS performance and maintenance” section.

There may be a current known issue that the customer is experiencing, with different troubleshooting steps.

Check known issues regularly.

3. Force the device to shut down and restart.

(Varies by new vs. old generation.)

Hold the power button for 10 seconds to force Windows to quit. Press once again start Windows. If that does not work, follow these steps by model:

Studio, Laptop, Pro 1796, Book 2, and Go: Press and hold the power button for 20 seconds.

Other devices: Press and hold the volume-up button and the power button at the same time for at least 15 seconds and then release both.

The Surface may boot to UEFI. This is normal. Have the customer exit UEFI.

Video available at Force a shut down and restart your Surface.

The device may be in a “hung” state. Force Surface to completely turn off all power and restart.

Performing a Forced Shutdown is like removing the battery from a laptop or unplugging a desktop PC. It resets the hardware and clears all the cache and memory for your hardware.

4. Run SDT or Windows Updates (WU).

For SDT: Have your customer go to http://aka.ms/surfacedt.

For WU: Go to Settings > Update & Security > Windows Updates > Check for updates

Note: Restart the Surface after installing updates.

Updates have fixes to common problems, or issues. Ensuring the customer has all the latest updates on their Surface will ensure they have the best possible experience.

5. Check Reliability Monitor.

Use Control Panel > Security and Maintenance > Maintenance > View reliability history.

View Critical events listed to gain problem insight. If events are app specific, troubleshooting should be performed on the app (remove and reinstall and/or contact publisher).

6. Scan for virus, spyware, or other malware.

Use Windows Defender to update definitions and Scan.

Virus, spyware, or other malware are a known cause of performance issues on PCs.

7. Check Task Manager.

From the Startup tab in Task Manager, disable unnecessary programs and then restart.

After the restart, from the Processes tab in Task Manager, view any items with abnormally high CPU usage. Click the CPU column heading to sort by it.

Check Windows stops responding on Surface for more info.

Unnecessarily running programs can cause slow system behavior and affect overall stability of the PC.

If issue is app specific, troubleshooting should be performed on the app (remove and reinstall and/or contact publisher).

8. Perform a Memory Diagnostic.

From Search, type Memory Diagnostic and run the Windows Memory Diagnostic program.

Check Windows stops responding on Surface for more info.

Errors with the PCs memory can cause unreliable system operation.

9. Reset

Good: Fresh Start

Go to Start > Settings > Update & security > Recovery.

Under More recovery options, select “Learn how to start fresh with a clean installation of Windows”. Follow the on-screen instructions.

Better: Reset from within Windows

Go to Start > Settings > Update & security > Recovery.

Under Reset this PC, select Get Started. Follow on screen instructions.

Best: USB recovery

The customer will need to download and set up a recovery image on a USB drive. Send the USB recovery link to the customer and help them get started. Information on what size and type of drive is provided at the destination page.

-OR-

Set up an order to have a USB recovery drive sent to the customer.

Fresh Start (good):

  • Fresh Start is the least intrusive form of reset.
  • Customers will not lose any files or personal information.
  • Customers will need to reinstall desktop programs.
  • Fresh Start also ensures the customer has all available updates once complete.
  • If there is an issue with Windows itself, the issue may still occur after a reset.

Reset from within Windows (better):

  • This option uses the devices Hard Drive Partition to reimage the device.
  • This option can delete personal data (the customer can choose Keep my files or Remove everything).
  • The customer will need to reinstall desktop apps.
  • The customer will need to run updates upon completion.
  • If there is an issue with the partition, the issue may still occur after the reset.

USB recovery (best):

  • This option reimages the entire device, including the partition on the hard drive.
  • It is the most complete form of reset.
  • Customers will need to back up their personal data.
  • Customers will need to reinstall desktop apps. Customers will need to run updates after completion.

If the issue occurs after a reset, then the root cause is very likely hardware related.

Is the issue resolved?

Yes, close the case: Once resolved, have the customer check for updates.

For SDT enabled sites, let the customer know about the Surface Diagnostic Toolkit.

For all other sites, check Settings > Update & Security > Windows Update for updates.

KB 4014334 Procedure: Surface: Using the Microsoft Surface Diagnostic Toolkit.

Making sure Surface and Windows are up to date may help avoid this problem in the future.

No, escalate the case: Work with a Tech Lead to make sure all necessary information is captured prior to escalation.

4. Windows - slow performance or high CPU Usage

Confirming the issue

While using the Surface:

  • Windows runs slower than expected. Windows may momentarily stutter on the screen and then resume. The Surface is still responsive to input, whether it be from the touch screen, trackpad, or keyboard but may be delayed.
  • The customer notices the CPU usage is high. Windows may also be running slower than expected or the battery not lasting as long.

Confirmation/paraphrase

"Just to make sure I understand your issue… "

Windows > Windows - Slow performance

“While you are using your Surface, it becomes slow to respond to input.”

Windows > High CPU Usage

“While you were using your Surface, you noticed in Task Manager that the CPU usage was higher than you expected.”

How to code

After confirming the issue, code the case to one of:

  • Windows OS > Performance and Maintenance > Performance - Windows > Windows - Slow performance
  • Windows OS > Performance and Maintenance > Performance - Windows > High CPU Usage

4. Check the power mode slider.

[Asset 4345697]

For improved performance, the slider should be set to Better or Best performance.

Setting the Power mode higher than Recommended will impact battery life, so the customer should connect to a PSU if possible or remember to lower the slider when they no longer need the CPU performance boost when on battery.

Surface devices have a configurable Power mode that lets the customer define their preference between Best battery life and Best performance.

The setting is configured separately for when the device is plugged and when it is on battery. This allows a customer to have this set to Best performance while on PSU and for it to automatically change to Recommended while on Battery.

6. Error - display driver stopped responding and has recovered

Confirming the issue

The display screen is momentarily frozen and then the error “Display driver stopped responding and has recovered” is displayed in the lower right corner.

Confirmation/paraphrase

"Just to make sure I understand your issue… When you are using your Surface, you see the error message 'Display driver stopped responding and has recovered.'"

Windows detects situations where the GPU did not update the display screen and the computer appears frozen. An updated video driver may help resolve this issue.

Best: USB recovery

The customer will need to download and set up a recovery image on a USB drive. Send the USB recovery link to the customer and help them get started. Information on what size and type of drive is provided at the destination page.

-OR-

Set up an order to have a USB recovery drive sent to the customer.

Fresh Start (good):

  • Fresh Start is the least intrusive form of reset.
  • Customers will not lose any files or personal information.
  • Customers will need to reinstall desktop programs.
  • Fresh Start also ensures the customer has all available updates once complete.
  • If there is an issue with Windows itself, the issue may still occur after a reset.

Reset from within Windows (better):

  • This option uses the devices Hard Drive Partition to reimage the device.
  • This option can delete personal data (Customer can choose Keep my files or Remove everything).
  • The customer will need to reinstall desktop apps.
  • The customer will need to run updates upon completion.
  • If there is an issue with the partition, the issue may still occur after the reset.

USB recovery (best):

  • This option reimages the entire device, including the partition on the hard drive.
  • It is the most complete form of reset.
  • Customers will need to back up their personal data.
  • Customers will need to reinstall desktop apps. Customers will need to run updates after completion.

If the issue occurs after a reset, then the root cause is very likely hardware related.

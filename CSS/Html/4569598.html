1. Overview

The versions of Microsoft Teams covered in this article include:

  • Microsoft Teams (for Personal): This is a free version of Teams that allows for easy switching of profiles between personal and work. At launch, it will only be available for mobile devices. A desktop version will be coming.
  • Microsoft Teams (Free): Is available, this is a free version of Teams intended to be used with organizations. It is available on desktop and mobile and is like the commercial version but has limited features. See here for more details.
  • Microsoft Teams (for Work): This is the licensed subscription version of Teams and is intended to be used in organizations. It has many features unavailable in the free version such as scheduled meetings, meeting recordings, and administration features.

Note: Teams for Personal is a new product that will continue to improve with new features. Please check with your Microsoft Technical Advisor if the feature list is not up to date.

System requirements

You can use Teams on these mobile platforms:

  • Android: Compatible with Android phones and tablets.

Support is limited to the last four major versions of Android. When a new major version of Android is released, the new version and the previous three versions are officially supported.

  • iOS: Compatible with iPhone, iPad, and iPod touch.

Support is limited to the two most recent major versions of iOS. When a new major version of iOS is released, the new version of iOS and the previous version are officially supported.

A desktop version will be available later. For more information on Teams requirements, see Hardware requirements for Microsoft Teams article.

Tool User Guide: Key Information Tool (KIT)

What is this article about?

This article outlines Key Information Tool (KIT) usage and product key checking.

 

1. Overview

Key Information Tool (KIT) is a web-based tool built on the KIS (Key Information Services) web service that provides product key information and activation details. KIT is used by Support Teams to understand general information on a Product Key, how the key is configured, and activation details. This information helps support teams understand problems with the use of a Product Key and how to proceed with troubleshooting.

2. What is a product key?

A 5x5 product key is a product identification value for a software assembly. The product key value comprises of 25 alphanumeric characters, grouped into five sets of five characters, which are separated by dashes, for example, XXXXX-XXXXX-XXXXX-XXXXX-XXXXX. A product key is assigned to an individual copy of a Microsoft software product and is used to install or activate the product. Product keys contain encrypted information including GroupId, and Channel ID/Serial No.

See KB 887343 Procedure: MPA: Product keys and Windows XP for more information on the types of product keys.

3. What is a Product Identification Number (ProductID or PID)?

A PID is a 20-digit product identifier and is extracted from a product key during installation.

See the Known limitations and FAQ article in the side navigation for more information on Product ID’s.

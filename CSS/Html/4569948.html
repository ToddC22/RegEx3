Procedure: Microsoft account: Top scenarios solution path

[Asset 4049349]

Who should use this article?

  • All Consumer

What is this article about?

This guided walkthrough leads users through the top Microsoft account scenarios for the best resolution.

  1. Confirm the customer is attempting to sign in to the correct account
  2. Log on to https://account.microsoft.com/ and select Forgot my password or reset it now.  (Alternatively, ask the user go to  https://account.live.com/password/reset) 

Does the user have access to the proof on the account?

  1. Ask the user to choose how they want to receive the security code (email or phone).
  2. Ask the user to enter the security code 
  3. Reset the password

  • KB 2984708 Procedure: Microsoft account: Two-step verification (TSV) contacts

 

Does the user have TSV enabled?

Once the account is verified, check the provided email address and select the link to reset the password.

If the user cannot verify their account, advise the user to try again (up to twice daily) and walk them through the success tips.

As an absolute last resort, advise the user to create a new account.

Customer education:

  • When we see activity in the user's account that looks like someone else might have gotten their password, we lock the account to prevent anyone from accessing their personal information.  
  • Customers are prompted to verify their account with a security code when first signing in after an account is compromised.

 Is the customer able to verify the account?

  • If customers can verify their account with a security code, they will see message 'help us secure you account' at sign in. Select Next to continue.
  • The customer will be asked to change the password then Next to continue.

The user will be asked to review their security info and remove or edit the information as necessary. 

Important: If there are any pieces of security info listed that a user does not recognize or can no longer use, they should select the checkbox next to that item to uncheck and remove it from the list of their security info. 

Note: If the user does remove security info, advise that they should sign in to http://account.microsoft.com after finishing this and replace removed security info with new security info. 

  • The user will receive confirmation that their account is now unlocked and they can sign in as usual with their new password.
  • After changing password, check the account to determine if there were unauthorized activities.

Customer Education: An account alias is an email address or phone number used to sign into your Microsoft account. Users can have multiple aliases, and use any of them with Microsoft services such as Outlook.com, Skype, OneDrive, Office, Xbox, Windows and more.

  1. Sign in to the https://account.microsoft.com/
  2. Select Your Info
  3. Select Manage how you sign in to Microsoft

Do you want to add or remove an alias? 

  • Select add email or add phone number. 
  • Add the phone number or email and select add alias. Users can create a new email address or add an existing third-party email. Note: You cannot add an alias that is assoicated with another account. 

Need more information?

  • Change the email address or phone number for your Microsoft account 
  • KB 4014162 Procedure: Microsoft account: Rename alias

 

Select Remove next to the alias you want to delete.   

Is it the primary alias? 

  1. If the user wants to delete the primary alias, they need to select another primary first. If the user signs in to Microsoft services with a primary alias, they may not be able to sign in to those services after removing the initial primary alias.  
  2. Select Make primary by the alias the user wants to set as the primary alias.
  3. The primary alias has been changed.  You now can remove the former primary alias. 
  1. Select remove by the alias you want to remove. 

Important:  If the user wants to remove an alias with a Microsoft domain, such as @hotmail.com, @live.com, @outlook.com, or @msn.com, it will be permanently deleted and no longer  associated with any Microsoft account.

  1. Before deleting, be sure to save any information you need from this alias. All emails and contacts for this alias will be deleted from the server immediately and cannot be recovered. You will not be able to access Microsoft services such as Xbox, Skype, and OneDrive if you used the removed alias.
     
  2. Select Remove

User tips to help them be successful:

  • Complete the questions on a device you’ve recently signed into your account on and a location like your home or office where you commonly use that device.
  • Outlook.com or Hotmail.com users, we’ll ask you questions about your contacts and email subject lines. Friends and family that you correspond with may be able to provide such subject lines.
  • If you purchased Microsoft services with this account before, please fill in the information below (Name on the CC, expiry year and month, expiry date maximum 1 year before).
  • If you have an Xbox console, you will be asked to provide the hardware ID of a console that you’ve used frequently. Make sure you collect the ID from the console you use most frequently. Visit the following link to learn how to collect the ID: https://support.xbox.com/help/hardware-network/warranty-service/find-xbox-one-kinect-serial-number
  • If you recently bought a new console, try to get the hardware ID from your old console. 
  • Skype users will be asked for the SkypeID or names of contacts on their account. 
  • If you use Skype to call mobile or landline phones, you’ll also be asked for some of those along with details about any recent Skype purchases. 

Customers who have Two Step Verification (TSV) enabled will not be able to recover their account by going through the Recovery Form. See KB 2984708.

You receive an error code 0x80070643 when you try to install Windows Live Essentials

BUG #: 118508 (Content Maintenance)
When you try to install a Windows Live program (for example, Windows Live Messenger or Windows Live Family Safety), the Windows Live Essentials installation fails, and you receive the following error code:
0x80070643
This issue occurs because a previous installation puts the computer in a state where a restart is pending.
To resolve this issue, restart the computer, and then reinstall Windows Live Essentials. To do this, follow these steps:
  1. Restart the computer.
  2. Visit the following Windows Live Web site:
    http://download.live.com
  3. Click the Download button.
  4. When the file download dialog box pops up, click Run to start the Installation Wizard.
  5. Reinstall Windows Live Essentials
Important Do not try to uninstall Windows Live Essentials before you reinstall it.

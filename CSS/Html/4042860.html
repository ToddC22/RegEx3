1. How to access

1.1 Complete and Pass Policy assessment

All users are required to complete and pass the knowledge assessment for the Tool Usage and Product Key Checking Policy. Providing proof of this, along with business justification including manager approval, is needed to gain access to the tool.

  1. Review the Tool Usage and Product Key Checking Policy within KB 4042868 TUG: KIT: Usage and product key checking requirements.
  2. Complete and pass the assessment. Use the link below and search for Course ID S1981618.
  • Delivery Partners, use the Success Factors here, and select Learning within My Info.
  • MS Internal, use Infopedia here.

1.2 Access Requests

KIT in RAVE:

Partners: Submit your request through your Sponsor Delegate.

Microsoft internal: Fill out the Non-Critical Issue/Request form and submit via https://o365supporthelp.azurewebsites.net/.

All users are required to access KIT in RAVE using Modern Identity (for example, example@office365support.com). Note: Your corporate domain account will not work.

1.3 Tool location in RAVE

Open RAVE https://RAVE.office.net/ and log in using your Modern Identify.

RAVE has two different business units: Microsoft 365 Consumer, and Microsoft 365 Commercial. Depending on which business unit you’re using in RAVE will determine where the tool will be located. To check, select your account icon [Asset 4566421]

1.4 How to Access KIT in RAVE

Once you’ve logged in using your Modern Identify account, how you access KIT is determined by how you use RAVE within each Business Unit:

  • I create or work cases in RAVE.
  • I use RAVE for case review or to access KIT only.

RAVE – Microsoft 365 Consumer Business Unit

I Create or Work Cases in RAVE  

Open a new case:

  1. Search using the customer’s Microsoft account in the “Search RAVE” field and fill out the required details to create your new case.
  2. Select on your new case number.
  1. Select the customer’s account in the upper left to open the profile to view tabs. 
  2. Select the Tools tab, then select KIT to open it.

Customer has no Microsoft account or alternative Email account:

  1. If searching customer’s assumed Microsoft account yields no results, customer does not have an Microsoft account, or does not have a personal email address, use this sample Microsoft account KITTool@contoso.com in the “Search RAVE” field.
  2. Select Create Ticket button in right corner. Fill out the required details and use this Microsoft account for Email: KITTool@contoso.com
  3. Select Assign to Myself, and then select Create Ticket to create your new case.
  4. Select your new case number.
  5. Select the customer’s account in the upper left to open the profile to view tabs. 
  6. Select the Tools tab, then select KIT to open it.

Open an existing case for a returning customer 

Search either using a case number or the customer’s Microsoft account in the “Search RAVE” field to open an existing case:

Using a case number:

  1. Select the customer’s account in the upper left to open the profile to view tabs.
  2. Select the Tools tab, then select KIT to open it.

Using an Microsoft account:

  1. Select customer’s account with the Type Microsoft account from the results list.
  2. Within Profile tab, search for case number within Support Case History section.
  3. Select the customer’s account in the upper left to open the profile to view tabs. 
  4. Select the Tools tab, then select KIT to open it.

I Use RAVE for Case Review or to Access KIT Only

Access KIT using Customer’s Microsoft account

  1. Search using the customer’s Microsoft account in the “Search RAVE” field and fill out the required details to create your new case.
  • For Support Area, select KitTool
  1. Select Assign to Myself, and then select Create Ticket to create your new case.
  2. Select your new case number.
  3. Select the customer’s account in the upper left to open the profile to view tabs.

  1. Select the Tools tab, then select KIT to open it.

Access KIT where Customer has no Microsoft account or alternative Email account:

  1. If searching customer’s assumed Microsoft account yields no results, customer does not have an Microsoft account, or does not have a personal email address, use this sample Microsoft account KITTool@contoso.com in the “Search RAVE” field.
  2. Select Create Ticket button in right corner. Fill out the required details and use this Microsoft account for Email: KITTool@contoso.com
  3. Select Assign to Myself, and then select Create Ticket to create your new case.
  4. Select your new case number.
  5. Select the customer’s account in the upper left to open the profile to view tabs. 

[Asset 4566419]

  1. Select the Tools tab, then select KIT to open it.

Access KIT without creating a case using a sample Microsoft account (restricted organizational access)

  1. If searching customer’s assumed Microsoft account yields no results, customer does not have an Microsoft account, or does not have a personal email address, use this sample Microsoft account customer@contoso.com in the “Search RAVE” field.
  2. Select account with Name Customer Contoso with Type Microsoft account from the results list. (There’s no need to do fortification.)
  3. Select the Tools tab, then select KIT to open it.

RAVE – Microsoft 365 Commercial Business Unit

You must create a case to access to KIT.

I Create or Work Cases in RAVE

Create a new case

  1. Search using the Tenant ID or Domain in the “Search RAVE” field and fill out the required details to create your new case.
  2. Either enter the new case number in “Search RAVE” field to bring up your case, or navigate to “My Cases”.
  3. Open KIT from the Troubleshooting section.

Open an existing case for a returning customer

  1. Search using the Tenant ID, Domain or case number in “Search RAVE” field.
  1. Open up the case.
  1. Open KIT from the Troubleshooting section.

I Use RAVE for Case Review or to Access KIT Only

Create a new case using a sample Tenant ID

If you’re unable to find a Tenant ID or Domain for the customer, or if they do not have one, a sample tenant needs to be used to access KIT:

  1. Search using this sample Tenant ID KIT@KITTool.onmicrosoft.com in the “Search RAVE” field.
  2. Select the Create new case button in right corner. Fill out only the required details with the following information:
  • Administrator Email: KIT Access – KIT @ KITTool.onmicrosoft.com
  • Customer Problem: KIT Tool
  1. Select Assign to Myself and select Create Ticket to create your new case.
  2. Copy your new case number, and OK.
  3. Either enter case number in “Search RAVE” field to bring up your case, or navigate to My Cases.
  4. Open KIT from the Troubleshooting section.

Once you’ve completed your troubleshooting session with that customer, document and close your case.

  1. Go to the Summary Tab and fill out the fields using the following:
  • Issue Description: KIT Tool
  • Bad URL: blank
  • Tags: blank
  • Resolution: Resolved
  • Resolution Steps: KIT Tool
  • Contact Outcome: Contacted
  • Verification: Verified by Agent
  • Did the proposed solution help: select the appropriate option (yes/no)
  • Select Save.
  1. Under +Actions In the right pane, select:
  • Case Status (internal): Troubleshooting
  • Case Action: Close Case
  • Select OK next to Case Action.
  • Confirm: Yes
  • Select Close.

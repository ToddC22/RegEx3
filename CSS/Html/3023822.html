Complex Install Scenarios: dual-boot/multi-boot, BitLocker, change boot order, RAID configurations NO NO NO PRIMARY NO NO

The configuration and setup of a RAID drive configuration are not supported.

This often requires documentation and knowledge of the third-party firmware software. However, if the RAID is configured correctly, this should NOT affect conventional installation routines. In other words, installation of Windows or Office on a device with a RAID drive configuration is the same as a PC without a RAID drive configuration.

Once the raid configuration is completed, Technical Support are expected to support these devices.

Some dual-boot/multi-boot scenarios that require changing drive boot order in BIOS require OEM support. Eligible customers may be redirected to OEM for additional help.

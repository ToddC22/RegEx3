Welcome to the Windows Networking Community Workflow site, for all aspects of Windows networking below the application, including transport of all network information, plus networking technologies in Windows such as DNS, DHCP, SMB, HNV, Hyper-V networking, WinHTTP, WebDAV, Wireless, 802.1x, Windows Firewall, VPN, RADIUS, and much more.

These are living documents – to add/ change/ improve any article, just send your changes (copy and paste original into Word and mark modifications) to the content owner/ team; that’s all you have to do. 
Also, if a topic you are interested in is missing or you would like to add one, please email.

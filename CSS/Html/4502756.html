2. Definitions

  • Ad - an individual online advertisement displayed to a potential customer.
  • Expanded Text Ads - mobile-optimized ad format that enables you to craft longer ad copy and optimize your ad text to better engage with potential customers before they click on your ads. Expanded Text Ads work seamlessly on mobile, tablet, and desktop devices, giving you a way to create more compelling calls to action for consumers and drive higher conversions to your business.
  • Upgraded URLs - Upgraded URLs are now generally available for all customers worldwide. It is not required to migrate all your existing URLs, which include standard text ad URLs, keyword destination URLs, and Sitelink Extension URLs, to start taking advantage of Expanded Text Ads. You are still able to manage and optimize standard text ads using destination URLs and set up new Expanded Text Ads using final URLs. We do recommend setting up an account-level tracking template to ensure that you can properly track and report your ad clicks.
    • URL parameters - URL parameters allow you to track information about the source of an ad click.
    • Custom parameters - Custom parameters works the same as URL parameters with respect to dynamic substitutions, except that you will define the parameter names and variables (also known as key and value pairs). You can define custom parameters for one or more campaigns, ad groups, ads, keywords, or Sitelink Extensions and then add them to the corresponding tracking template.
  • Countdown - Countdown customizers let you easily add a countdown — by day, hour, and then minute — to an event in your Expanded Text Ad. The countdown, which automatically updates as the event draws nearer, is eye-catching and gives potential customers greater incentive to click your ad.
  • Param - This defines a custom text for each keyword when using dynamic text. Creating Param2/Param3 insert values is out of scope for the BOS team.

3. Best practices and policies

3.1 Best practices

  • Ads are created on the ad group level.
  • Ads should be related to the context and structure of their respective ad group and/or campaign.
  • You can add a maximum of 100 ads combined for both active and paused ads of all types at the ad group level. See Entity Hierarchy and Limits for details.
  • You can create Ads using static or dynamic text.

Return to top

3.2 Policies

  • Ads may not use excessive or random capitalization.
  • Symbols like emoticons or block elements such as ▉ or ▃ are disallowed.
  • The Ad URLs need to match the domain used in the ad group.
  • The combined ad title and ad text must be at least three words long.
  • The content must adhere to the Microsoft Advertising content and style policies.

5. Dynamic text ads

Dynamic text is a word or phrase that is automatically inserted into the ad title, ad text, or display or destination URL. Using dynamic text helps create more relevant ad copy and customizes ads to a user's actual searched term. This can help get more clicks and conversions.

Important notes:

  • For the Ad Copy Writing service, display and destination URLs will come from the existing ads. Advocates should NOT create new URLs.
  • When implementing dynamic text in your ad copy, you should provide a default string for example {Param1:default} that the system will use if Param1 for a keyword is null or empty, or if including the Param1 substitution value will cause the expanded string to exceed the element’s limit; otherwise the ad will not serve.
  • Keywords in an ad group are inserted into an ad title and/or ad text when the placeholder: {keyword} is typed into the ad title and/or ad text. This is called keyword insertion.
  • In addition to dynamically inserting the keyword, you can also add some custom text for each keyword.

Keyword capitalization options for dynamic texts

Capitalization options for keyword placeholders help you to make sure your ad always looks great to your customers.

Suppose your business delivers fresh fruit and vegetables that customers order online, and you want each keyword to be capitalized in your ad title no matter how a person types it in a Bing, AOL, or Yahoo search. Capitalizing the placeholder as {KeyWord} makes fresh fruit, Fresh Fruit, and even FRESH fruit always appear in your ad title as Fresh Fruit.

Note: The capitalization options apply only to the ad title, ad text, and display URL for campaigns and ad groups running in the following languages: Danish, English, Finnish, French, Italian, Norwegian, Portuguese, Spanish, and Swedish.

Key Points in Creating Ad Copy

Create Ad Copy that reference the points you have identified in the existing ad Copy and the customer’s website.

Note: Use the information gathered in Creating Ad Copy to focus your ad copy suggestions on services which already appear to be successful. Keep in mind the following rules:

  • You should write two-three ads per ad group being requested. At least two of these ads should be static.
  • If the advertiser is NOT utilizing dynamic text, only write static ads.
  • If the advertiser is already using dynamic text, write dynamic ads utilizing param values.
  • Validate that dynamically inserted items make sense in the context of your ad copy. Use Default text to avoid character limits {param2:defaulttext}.

Review the following key elements which should be incorporated wherever appropriate and applicable to your newly created ads:

  • Brand: If you have a well-known brand name, use it in the ad title and add a trademark symbol or ‘’official site”. Shoppers trust a brand they already know and appreciate.
  • Emotion: Think about what moves the customers and use that to build a connection.
  • Call to Action: Create a sense of urgency by adding strong callouts like “Buy Now”.
  • Incentives: Include any discounts, shipping offers or seasonal sales like “Save 25%” Shoppers will always look for the best bargains.
  • Fear/Pain: Think about what customers fear or what they would miss out on like “Sale ends 9/10”.
  • Advertiser Tone:  Check out the existing Ad Copy and landing pages…try to match your style of language to meet the advertiser’s own, if possible.
  • Special Requests:  Make sure that you read and implement any special requests that the requestor makes, in case there is a particular type of ad they are trying to create.
  • Address your customers directly: Use the words "you" or "your" in your ads so that you are speaking directly to them. You know what we're talking about.
  • Pre-qualify your visitors: Use words in your ads that help attract true potential customers. For example, if you offer only high-end products, stay away from words like discount, bargain, and cheap. Be clear in your ad who your products are for. This can save you money by eliminating clicks that are not likely to convert to sales.
  • Be specific: The clearer and more specific your offering, the better. For example, rather than "big discounts," specify an exact percentage, such as "50% off."
  • Describe what sets your product apart: This is just Advertising 101. Why do customers buy your product? Does it solve a problem? Does it help save time or money? Or, does it have unique features or qualities? For example, if your product is available in many more colors than the competition can offer, you might mention color choices in your ad.
  • Keep your ads relevant: One way to improve relevance is to use your customers' most popular search query keywords in your ad titles and text.

Note: Make sure your suggestions comply with Microsoft Advertising editorial guidelines. You may also refer to What makes an effective ad? for this.

13. FAQ

Q: The submitter has a provided a list of existing ad groups for ad copy creation, and it is a bulk request. Do I need to check for their performance before I create ad copy?

A: If this is a bulk request and would exceed the standard SLA, check for the top 25% performing ad group first and choose whichever if feasible. Ask your Support for suggestions and get your submitter’s approval before proceeding.

Q: Is there a character limit for Final URL and Mobile URL?

A: Yes. There is a 2,048-character limit each.

Q: Are Upgraded URLs required in order to use Expanded Text Ads?

A: No. Upgraded URLs are now generally available for all customers worldwide, but it is not required to migrate all your existing URLs (such as standard text ad URLs, keyword destination URLs, and Sitelink Extension URLs) to start taking advantage of Expanded Text Ads. You are still able to manage and optimize standard text ads using destination URLs and set up new Expanded Text Ads using final URLs. We recommend setting up an account-level tracking template to ensure that you can properly track and report your ad clicks.

Q: Will standard text ads be migrated to Expanded Text Ads?

A: No, there is no migration from standard text ads to Expanded Text Ads. Expanded Text Ads are a new, mobile-optimized ad format that enables you to craft longer ad copy and optimize your ad text to better engage with potential customers before they click on your ads.

Q: Can I use {param1}, {param2}, and {param3} for keyword tracking in Expanded Text Ads?

A: Yes, you can add a {param} to a domain in the Final URL box, but you cannot have your Final URL be only a {param}. In other words, in the Final URL box, “contoso.com/{param1}” is allowed, but just “{param1}” is not. {param1}, {param2}, and {param3} are also still allowed in ad titles and descriptions.

Note: Keep in mind the length of the potential dynamic text a {param} might cause. It is possible to create ad titles, ad text, and URLs that fit within our character limits when you create them, but they can become too long when the dynamic text is applied. In this case, the ad will not be considered for auction.

List of all 2007 Office system Service Pack 2 and Windows SharePoint Services 3.0 Service Pack 2 packages

BUG #: 110257 (Content Maintenance)
This article lists all the 2007 Office system Service Pack 2 (SP2) and Windows SharePoint Services 3.0 SP2 packages and information about how to obtain the SP2 packages.
To obtain these SP2 packages, use one of the following methods.

Method 1: Microsoft Update (recommended)

To download the service pack from Microsoft Update, visit the following Microsoft website: 
http://update.microsoft.com/microsoftupdate/v6/default.aspx
Enrolling in Microsoft Update is the recommended way to update the products to SP2. Microsoft Update will detect the products that you have installed and then apply all updates to those products.

Method 2: Download the SP2 packages from Microsoft Download Center

All SP2 packages are available for the download from the Microsoft Download Center. You can download and install the following service packs for each product that you have installed.

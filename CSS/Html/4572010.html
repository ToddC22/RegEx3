Tool User Guide: Partner: Service Desk Case

[Asset 4090831]

 Summary 

Who should use this article 

All Partner Frontline advocates supporting Partner Center, using Service Desk Case (Globally)

What’s the article about 

This article provides a list of Service Desk Case Knowledge (SDK) resources to help Partner Frontline advocates understand how to effectively use the Service Desk Case tool when handling Partner support requests.

Note

Articles contained within this document are primarily focused on the use of Service Desk Case. Addition process related articles for Partner engagement may be found by searching topics on the Service Desk Knowledge site.

 

Service Desk Case Articles

Tool User Guide: Partner: Service Desk Case (This article)

Brief program overview with links and descriptions of Partner Service Desk Case articles.

Tool User Guide: Partner: Service Desk Case Environment

Introduces the tiles and views available for advocates within the Service Desk Case dashboard.

Tool User Guide: Partner: Manual Service Request Creation in Prose (PCST) 

Instructions for advocates who need to create a case in Prose after receiving multiple issues in a single online Support Request, phone call or email from a Partner. 

Tool User Guide: Partner: Service Desk Case Basic Case Management

Instructs advocates how to perform basic case management features including viewing, editing, changing status, changing states, saving and closing cases.

Tool User Guide: Partner: Service Desk Case Partner Communication

Instructs advocates how to communicate and log email, web communication submissions, phone and strike counts during case management.

Tool User Guide: Partner: Service Desk Case Collaboration and Follow-up Tasks

Instructs advocates how to use collaboration and follow-up tasks within Service Desk.

Tool User Guide: Partner: Time Tracking with Case Buddy

Instructs advocates how to track billable time for cases handled inside and outside Service Desk Case. 

Tool User Guide: Partner: Service Desk Case Administrative Reroutes, Misroutes, Signposting, Handovers, Returning Cases, Disconnects, PII Failed and 715 Error Codes

Instructs advocates how to transfer or handoff cases to other Service Desk enabled support groups or misroute cases which cannot be transferred outside Service Desk. In addition, advocates will learn how to handle cases when a Partner receive a 715 Error code.

Guideline: Partner: Transfers and Escalations for Partner Frontline Support using Service Desk 

Provides advocates visibility into the resources and tools to utilize when performing case escalations and transfers. 

Procedure: Global: Contacting Global MPN, CSP and Commercial Marketplace Support teams in Service Desk

Provides information to other teams on how to engage us for scenarios in scope for MPN, CSP and Commercial Marketplace.

Guideline: Partner: Case Management Checklist

This article provides an advocate who is familiar with Service Desk Case, a simplified checklist for common case management scenarios.

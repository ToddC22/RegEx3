1. Applies to

The following services: Microsoft 365
The following advocate roles: All roles
The following markets: All markets (For detailed information about product-specific availability, see article: 2748142: Markets where Microsoft Online Services are available .)
 

Scenario
 

A customer submits a case or calls for requesting to help reviewing or editing information in the Microsoft Admin Center (Classic) or Microsoft Admin Center (Default). 

back to the top

2. Guidance

The Country or region field cannot be edited, because it is permanently fixed with the tenant. If the customer requires an address in a different country, they will need to create a new tenant.

Welcome to Intune's workflows for Apps-Deployment. Here you will find all you need to know about working these types of cases.

In order to better understand Application Deployment, it is necessary to understand what is an application.
It is basically a software program running on a smart phone, tablet, Computer, Virtual Reality device, or even in the cloud. For example: Calculator, Outlook, Word, Facebook, Edge,..
 
Within Intune, these application are categorized in different types depending on their type, hosted location, etc. For example:
  • Store apps: these are applications hosted at various stores like Apple Store, Google Playstore, Microsoft Store.
  • Line of Business Application (LOB): These are usually applications not available for retail distribution and can be either loaded with the actual file but can also be hosted in a private store.
Each of these apps are platform specific (iPhoneOS, iPadOS, Android, Windows, Mac) and within Intune, each has limitations as to which application type is supported and what is required in order to install them. For more info see https://docs.microsoft.com/en-us/mem/intune/apps/apps-add

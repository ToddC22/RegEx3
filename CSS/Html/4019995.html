1. Overview

Virus and Malware removal is part of the Microsoft paid services (Assure, Microsoft Complete and PPI) and PC or Tablets purchased at Microsoft Store.

Your primary and only tool for scanning and removing viruses and malware is Windows Defender AV and Windows Defender Offline.

The use of the Answer Desk Virus and Malware Removal tool (MSDT) is no longer permitted and has been removed. You should no longer use any of the tools that were listed in the Answer Desk wrapper, including third-party tools such as Hitman Pro. Third-party tools for virus and malware removal are not permitted.

Manual removal of virus and malware is not recommended nor permitted. If a security threat (for example, virus, and malware) has been detected, and Windows Defender Offline is not able to remove it, advocates are expected to recommend a re-install of Windows or a reset/refresh of Windows. This approach is the fastest, most efficient way to recover a system compromised by malware infection.

You are not required to stay connected with the customer for the duration of the virus/malware scan. The time required for the scan and removal process varies, as it depends on the number of files on the PC and the hardware specification. If the customer requests a follow up based on the outcome of scan/removal process, advocates are expected to proceed in accommodating a scheduled callback (escalation for a follow up). The callback arrangement (escalation) should only be made based on the customer's request. Only if customer request to stay connected, you can do so but be mindful of time you spent as well, you can always suggest calling back if you expect the process takes long time.

Avoid unnecessary troubleshooting steps. Numerous steps that are considered inappropriate have been identified. Many of these methods are very time-consuming and are inappropriate based on the call type (symptoms). Below is a list of troubleshooting step examples and approaches that should be avoided for a virus and malware removal:

  • Clearing Pre-fetch
  • Deleting cached files
  • Deleting temp files
  • Clearing Event Viewer
  • SFC scan
  • Defrag
  • Disk cleanup
  • Installing third-party ad blockers for web browsers.
  • Using Malwarebytes, or other third-party tools.
  • Staying online when running scans.

This is not limited to the list above. The goal is to eliminate inappropriate troubleshooting practices and make use of this article as the only resource for virus and malware removal.

3. Prerequisites

The customer must meet the service requirements before you begin.

Minimum requirements:

  • Microsoft account one-time offers, including PSS PC tune-up, and virus removal may be purchased without Microsoft account, using Guest Checkout. See KB 3039170 Procedure: Microsoft software support services refunds.
  • Broadband Internet connection.
  • Windows 7 or higher.
  • Internet Explorer 11 or higher, or Firefox 3.6 or higher.
  • 300 MHz or higher processor clock speed recommended, 233 MHz minimum required (single or dual processor system).
  • Intel Pentium/Celeron, or AMD K6/Athlon/Duron, or compatible processor recommended.
  • 256MB RAM or higher recommended (minimum might limit performance and some features).
  • SVGA (1024 x 768) or higher display.
  • Recommended use of default browser security setting levels.
  • Adobe (Macromedia) Flash 8 and higher.

5. Troubleshooting steps by scenario

5.1 ﻿Removing pop-ups

Edge: Malicious pop-ups generally appear when the customer is browsing the web. Once the pop-up appears, it won’t go away with normal procedures, such as close button or Alt + F4. When you do manage to close the pop-ups, sometimes it continue to re-appear on opening the browser.

Example (the text in the screen is designed to bring awareness to the customer and call a fake Microsoft number to encourage customers into paying money to remove it):

[Asset 4020525]

To remove this type of persistent screen, follow these steps:

  1. Close the windows with Task Manager. Select CTRL+ ALT + Delete keys simultaneously, which will bring you to this screen:

 [Asset 4020526]

  1. Select Task Manager, which will bring up this screen:
 

[Asset 4020527]

  1. Select Microsoft Edge in the list, and then End task. This will close the Edge program.


Note (Creators update only): After closing and re-opening Microsoft Edge, you will start at a new page, as Edge knows it was terminated. The offending page will be on its own tab and can be closed. If pop-up is resolved in this step, you can skip the remaining steps.

  1. Next, remove the Edge package from the operating system and replace it with the default version that ships with Windows 10. The MSDT tool Edge Malicious Pop-up removal tool doesn't remove virus or malware; it reinstalls and resets Edge.

[Asset 4020528]

Return to top

5.2 Manually reset Edge

If the MSDT tool Edge Malicious Pop-up removal tool fails to execute, manual steps can be taken. Just like MSDT tool Edge Malicious Pop-up removal tool, this does not remove virus or malware, this reinstalls and resets Edge.

Open the PowerShell ISE and run the following command:

remove-item$env:localappdata\Packages\Microsoft.MicrosoftEdge_8wekyb3d8bbwe\* -recurse-Force 2>$null

5.3 How to use Defender Offline with third-party Antivirus installed?

When a third-party antivirus is installed, such as Norton, Defender signatures won’t get updated and Defender Offline is not enabled.

[Asset 4020529]

[Asset 4020530]

You can turn Defender back on without removing third-party antivirus by turning on Limited Periodic Scanning. In Windows Defender, select Turn On, or in Settings > Update & security > Windows Defender, and turn on Limited Periodic Scanning.

[Asset 4020531]

Norton will show a pop-up dialog suggesting the user to turn Limited Periodic Scanning off.

[Asset 4020532]

For more information, see the Learn More page.

After turning on Limited Periodic Scanning, you need to download the latest Signatures through Windows Update or Windows Defender.

[Asset 4020533]

Then you can run Defender Offline.

[Asset 4020534]

7. FAQ

Q: Why Windows Defender or Windows Defender Offline?

A: Windows Defender is underestimated in its capability and effectiveness, especially the "Offline" version. The primary benefit of using this tool is that it runs before malware, such as rootkits, can hide. When you perform a post-event malware scan and remediation by running the scan on a system infected with advanced low-level malware, the malware has a chance to run first. The malware itself may be intercepting the antimalware software’s attempts to inspect, take actions, or communicate to the user. When you run an “offline” tool like Windows Defender Offline, you’re bringing your own known-good, clean operating environment with you along with the scanner.

Q: Can we still escalate these cases?

A: Yes. However, we are anticipating a reduction to the escalation rate since manual removal of malware is no longer allowed. Moreover, an escalation might be a good choice if the customer is requiring additional assistances to reinstall the OS later.

Q: I used Windows Defender, and it did not detect any malware or was not able to remove the malware. Can I go back and use another tool like Hitman Pro?

A: No. If malware was not detected, it's possible that the root cause of the issue might not be malware related. Therefore, it's important not to communicate or act on poor assumptions based on the symptoms of the issue. If Windows Defender detected malware and it was not remediated, Windows Defender Offline will have a high probability for remediation.

Q: What's the difference between Windows Defender AV and Windows Defender offline?

A: Windows Defender is an antivirus that is included in the latest version of Windows (Windows 10). For other version of Windows, it's available as a download. Unlike the offline version, it runs on Windows. For more complex threads such as a rootkit, Windows Defender may not be as effective. Since Windows Defender works "offline", there are less changes where remediation is blocked by the malware.

Q: Is Answer Desk paid offering still required (for example, Assure Support)?

A: Yes.

Q: What if the customer declines the paid offer?

A: You can share this URL with the customer on how to use Window Defender Offline as a self-help resource: https://support.microsoft.com/help/17466/windows-defender-offline-help-protect-my-pc

1. Search for customer and create case

In this section, the agent has no information about the customer and needs to be able to:

4. Basic payment search

[Asset 4503351]
Select Payment Search to open the Payment Search Window

[Asset 4132163]

Search by the Name on the Customer's card, the last 4 digits of Credit Card, and Expiration Date of the card.

Note: All three entries are required to perform a valid search. The name will match what the customer entered when they added the card, not necessarily what is on the card itself.

[Asset 4132164]

Select Full Credit Card to search for the customer's complete credit card number

Return to top

6. View all 
 

All incidents ever reported by the customer are listed in this area, both Open and Closed.
Select View all to see the entire list. Click on the SR# to open a particular SR and take action.

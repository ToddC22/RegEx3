This is a tip which helps you to remove the music tracks within Media Center part but without deleting them from the computer.
1]Open Windows Media Player.
2]Click on Library tab.
3]Select All Music present in the left pane.
4]Select one music file.
5]Now press [CTRL] + [A] to select all the music files.
6]Press the Delete key.
7]You will get a pop up like this:-
Are you sure you want to delete the (X no) selected items?
  • Delete from library only
  • Delete from library and my computer
8]Select the first option i.e.' Delete from library only.'
9]Click OK.

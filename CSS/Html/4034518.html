Download-related Escalations for VL

For download-related escalations for VL, please also include

1. Live-ID:
2. Date Of last download attempt:
3. Time of last download attempt?
4. Product name:
5. Download file name: (ex: SW_CD_Office_Professional_Plus_2007_W32_English_MLF_X-16-19797.ISO)
6. User’s country Location?
7. Browser and browser version:
8. Are you downloading from behind a Proxy server (i.e. At work):
9. Do you have any Proxy restrictions (ask system admin)?
10. Provide us the full screenshot of the page where the failure occurs including a screenshot of the Download Manager error if applicable.
11. Identify your external IP address by navigating to: http://whatismyip.akamai.com
12. In a command window type nslookup vl.download.microsoft.com (provide screenshot)
13. In a command window type tracert fullproduct.download.microsoft.com (provide screenshot)
14. Identify the Operating System, browser, and Java Runtime Environment of the user.
15. Provide a screenshot of the LAN settings by going to the following path in Internet Explorer: Tools > Internet Options on the Connections tab under LAN Settings MSPP Support issue:
16. Identify your DNS IP Address by navigating to: http://www.damia.com/whatismydns

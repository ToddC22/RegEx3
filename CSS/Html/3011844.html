Policy 

Compute core increases for Batch accounts are handled by the Quota Automation Service (QMS).

Customer Experience and Provisioning 

How to Access Batch Services 

Customers can access Batch services by following the below steps: 

  1. Navigate to portal.azure.com. 
  2. Click on All services in the left blade. 
  3. Type Batch into the search box. 
  4. Select Batch Accounts.

Support Scenarios 

How to Check Current Batch Quota 

Batch quota usage can be checked by completing the following steps: 

  1. Navigate to  Jarvis.
  2. Filter for AzureBatch.
  3. Select AzureBatch.
  4. Select Quota Operations.
  5. Select Get Account Quota.
  6. Enter the Batch Account into the provided field.
    • This is the name of the customer's Batch account.
  7. Select Region from the provided list.
    • This is the region in which the customer Batch account is located.
    • If you are unsure which region the Batch account is located in, it can be looked up in Azure Support Center (ASC) using the following directions:
      1. Click the Lauch ASC button from your case in Service Desk.
      2. Navigate to the Resource Explorer in ASC.
      3. Locate the subscription and look for batchAccounts in the subscription.
      4. In the batchAccount's properties, the region is available under Location (Region).
  8. Click the Run button. 

Usage Verification 

If you need to verify your customer’s usage, engage the DOTD team through IcM. Please fill out all the information requested in the IcM form. 

To find the default and maximum quotas, see the Azure Subscription Service Limits page.

Batch Service Increases

If a customer is requesting an increase in the number of Batch Accounts, engage the DOTD team via an IcM to increase the Batch Accounts. To create a ticket, go to http://aka.ms/AzurBatch and fill out all the information requested in the IcM form. 

Note: Do not engage CM 24X7 via the RDQuota ticket process to increase the number of Batch Accounts. The Batch Engineering team owns this function, and no other team can handle this function.  

Jobs and Pools Quota Increase 

If your customer needs an increase in their Jobs or Pools quota, engage the DOTD team through IcM. Please fill out all the information requested in the IcM form. 

To find the default and maximum quotas, see the Azure Subscription Service Limits page.

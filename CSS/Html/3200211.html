Scenario
 

A customer with an Invoice Pay order requests a Statement of Account (SOA), account reconciliation, or their order balance.

3. Procedure

i. Create an assistance request for Tier 3 (Tier 2 for Japan) asking for a SOA and include the Account Balance Confirmation/Account Reconciliation Template in the Rave Internal Notes:

Account Balance Confirmation/Account Reconciliation Template

Company Name    
Order Number(s)  
Reason for Account Balance Confirmation, Statement of Accounts/Account Reconciliation  
Invoice Number(s) found with issues  
Customer Contact Name  
Customer Contact Email  
Customer Contact Phone Number  


*** For Model 21 RFI support, please add below fields in addition to above template before submission:

Customer Full Address  
City/State/Province  
Postal Code  
VAT number  



ii. Tier 3 (Tier 2 for Japan) will provide account activity for the past 12 months in the following format in step iii. If the customer needs account activity older than 12 months, advise Tier 3 (Tier 2 for Japan).

Tier 3 (Tier 2 for Japan) will check if the amounts in CMAT and SAP match.
a. If they match, Tier 3 (Tier 2 for Japan) will add the balance information or create the Statement of Account and attach it to the case.
Note: Tier 3 will not tag the tenant for DNC. If DNC is needed, create another new Assistance Request asking Tier 3 to process a PTP or POP (Proof of Payment) request.

b. If they do not match, Tier 3 will tag DNC for 30 days. Tier 3 (Tier 2 for Japan) will update Tier 1 and escalate the assistance request for the CAICAT support area. Tier 1 will then monitor the CAICAT assistance request for resolution. For any follow up questions about the SOA provided follow up with CAICAT team. 
iii. Tier 1: Send the Statement of Account to the customer. (Remove the fields in yellow before sending to the customer: Column A and Row 2.)  


Balance that match in SAP and CMAT but not in the portal, create a request to Tier 3 to reach out to pay-ops following the article 2778091: How to escalate a Service request to Tier 3, Engineering.

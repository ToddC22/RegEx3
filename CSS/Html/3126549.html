What is this article about?

This article provides information around OneDrive support boundaries and scenarios for Windows, Office, Billing, and Microsoft account.

Use this as a reference guide to determine the scope of support for customers with questions about OneDrive.

2. Scenarios

2.1 OneDrive Windows technical support or Office scenarios

Scenario

In-product email support

Office & Windows

All Others

Refence/Notes

Questions about:

  • Microsoft 365 subscriptions

OneDrive storage

PRIMARY

NO

All technical issues outside of quick and easy general questions: transfer to in-product support.
  • Chat and Phone advocates will direct customers to in-product email support. These include:
    • Microsoft 365 subscribers
    • OneDrive storage subscribers

See below for ASD ticket handling for when you need to transfer to in-product email support.

OneDrive Personal Vault

PRIMARY

YES

All technical issues outside of quick and easy general questions: transfer to in-product support.

Office and Windows advocates should be able to answer general Personal Vault scenarios such as:

  • What is Personal Vault?
  • How do I lock/unlock this feature?
  • How do I disable the feature?

Issues involving network, features not working as expected, or involve technical troubleshooting, direct to in-product email support.

See this article for more info on using Personal Vault.

OneDrive PC folder backup

PRIMARY

YES

All technical issues outside of quick and easy general questions: transfer to in-product support..

Office and Windows advocates should be able to answer general PC folder backup scenarios such as:

  • What is PC folder backup?
  • How do I setup this feature?
  • How do I change settings for this feature?

Issues involving network, features not working as expected, or involve technical troubleshooting,  direct to in-product email support.

See this article for more info on using PC folder backup.

Find missing files

PRIMARY

YES

All technical issues outside of quick and easy general questions: transfer to in-product support.

See this article: Find lost or missing files in OneDrive

4. How customers can get email support

Web

  • Login to OneDrive.com
  • Select the ? help icon on the top banner of the page
  • Select Email OneDrive support
     

iOS

  • Open OneDrive app
  • Select Me
  • Select  Help & Feedback
  • Select: Ask for help
     

Android

  • Open OneDrive app
  • Select the  Menu button
  • Select the  ? button
  • Select  Report a problem

UWP app (Windows Desktop)

  • Open OneDrive app
  • Select  Settings 
  • Select  Give Feedback (Help & Feedback)
  • Select the following Contact Us option: Ask for help
  • Select the  in bottom right corner

When you try to open a fax that is a TIFF file in Microsoft Office Document Imaging, the fax opens in the Windows Picture and Fax Viewer instead

Template: zGeneric Problem Article
When you try to open a fax that is a TIFF file in Microsoft Office Document Imaging, the fax opens in the Windows Picture and Fax Viewer instead. This issue occurs when a 2007 Microsoft Office suite is installed on the computer.
This issue occurs because TIFF files are not associated with Microsoft Office Document Imaging.
To resolve this issue, follow these steps:
  1. Right-click the TIFF file that you want to open.
  2. Point to Open With, and then click Choose Program.
  3. In the Programs list, click Microsoft Office Document Imaging.
  4. Click to select the Always use the selected program to open this kind of file check box, and then click OK.
If Microsoft Office Document Imaging does not appear in the Programs list, you may have to install Microsoft Office Document Imaging.

To install Microsoft Office Document Imaging in Windows Vista, follow these steps:
  1. Click Start, type Programs and Features in the Start Search box, and then click Programs and Features in the Programs list.
  2. Click the 2007 Microsoft Office suite that you are running, and then click Change. For example, click Microsoft Office Premium 2007, and then click Change.
  3. Click Add or Remove Features, and then click Continue.
  4. Expand Office Tools, and then click Microsoft Office Document Imaging.
  5. Click Run all from My Computer, and then click Continue.
  6. Click Close to exit the 2007 Microsoft Office suite setup.
To install Microsoft Office Document Imaging in Microsoft Windows XP, follow these steps:
  1. Click Start, type appwiz.cpl in the Start Search box, and then press ENTER.
  2. Click the 2007 Microsoft Office suite that you are running, and then click Change. For example, click Microsoft Office Premium 2007, and then click Change.
  3. Click Add or Remove Features, and then click Continue.
  4. Expand Office Tools, and then click Microsoft Office Document Imaging.
  5. Click Run all from My Computer, and then click Continue.
  6. Click Close to exit the 2007 Microsoft Office suite setup.

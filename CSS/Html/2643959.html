Who should use this article?

What is this article about?

This article contains information on all current promotions in market, organized alphabetically by promotion name. Offer values and expiration dates are specified in the quick link titles. Select the linked title to see more details about each offer. Actions related to these promotions are reactive (only if a replacement code is necessary), but you should also recommend customers sign up for email alerts to get future offers/coupons.

Note about markets and codes: If a customer meets all criteria for the promotion as listed below (you are responsible for verifying if criteria are met), but there are no codes for that specific currency amount on the SharePoint, you are authorized to give a code in a comparable amount of another currency so we don’t delay the closure of the ticket (for example, if there are no 100 Euro codes available, a $100 code may be issued if it is available).

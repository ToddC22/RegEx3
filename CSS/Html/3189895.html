ASMS Distribution Lists

Use the following table to find the appropriate ASMS distribution list (DL) based on your query.

Distribution List Name

Description and Use

ASMSACCESS
(Tools Access Governance)

Use this alias to request access to tools.

ASMSCI
(Case Insights)

Use this alias to contact the Case Insights team.

ASMSGETHELP
(Get Help)

This alias is for the GetHelp Azure Focus team to request escalations for support cases.

ASMSSP
(Support Programs)

Use this alias to share information with the Support Programs team or to ask questions that could not be answered by a TA.

ASMSTA
(Technical Advisors)

This alias is for TA communications only.

ASMSTT
(Trainers)

Use this alias to communicate with the Training team.
If you are a TA, PTA, or manager with a change request for training, please submit a TrainTrack ticket.

ASMSWRAPUP
(Wrap Up Emailer)

This alias is used by Support Programs to send out the Weekly Wrap Up email.

WASMSESC
(Escalations)

See TSG 4015188 for guidelines on when to use this alias.

WASMSGLOBAL
(Global Delivery)

This alias is for the management team to communicate to the global organization.

WASMSLEADS
(Leads)

This alias is used by TAs and PTAs to raise topics that need manager awareness or action.

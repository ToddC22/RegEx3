Who should use this article?

What is this article about?

This article provides which LOB supports what scenario for customers with questions or issues related data backup and recovery.
Use this as a reference guide to determine the scope of support for customers with questions about their data. "Data" is defined by customer's personal data such as pictures, music, and document.

  • Recovery Key – Bitlocker
 PRIMARY NO YES NO

Set up for personal use (using a Microsoft account)

The recovery key is automatically saved to that Microsoft Account as part of the automatic enablement of BitLocker. The customer will need to use the Microsoft account that was first used on the device and sign in to https://account.microsoft.com/devices/recoverykey to locate their recovery key.

As a backup, try searching the older site, https://onedrive.live.com/RecoveryKey, for the recovery key.

Set up for personal use (using an offline account)

When the customer enables Bitlocker, they will be asked to save the recovery key. There are 3 choices they can make that will determine where the recovery key is stored:

  • Save to your Microsoft account. For this option to be available, the customer needs to sign in with a Microsoft account instead.
  • Save to a file. The customer can save the recovery key to file that is not on the drive being encrypted. For example, a USB flash drive.
  • Print the recovery key. The customer can print the key to sheet of paper. If they did this, they need to locate that print out.

Set up for an organization

If the customer selected "Set up for an organization" or the device was already connected to their company’s domain, their recovery key may not be listed under their Microsoft account. Redirect the customer to an IT Administrator for the organization or company the customer belongs to.

Technical Guide: Surface: Surface Bluetooth Keyboard issues

[Asset 4049349]

Who should use this article?

  • Surface

What is this article about?

This article will help you address Bluetooth keyboard issues, including Bluetooth connectivity, power, clicking, scrolling, how to and setup, and other Bluetooth keyboard functionality. Use this article any time you work with a customer to resolve their Bluetooth keyboard issue. This can be used for Surface, Microsoft, or third-party Bluetooth keyboard troubleshooting guidance.

Resolving the issue

Once you have finished Confirming the issue, follow the resolution steps in the table below.

Follow the steps in the order below until the issue is resolved or until all the steps have been completed. While following the steps, use the "Why" column to inform the customer why you are performing the step and, if successful, to better understand what the cause may have been. As you follow the steps, document the result of each one in your notes. Then proceed to the "Is the issue resolved?" section at the end of the table for next steps.

Step

How

Why

1. Copy and paste the template to the right to begin your note taking for this scenario. Document your findings for every step performed.

Notes template:

  • Check known issues.
  • Verify the customer’s PC is compatible with the keyboard.
  • Restart the host.
  • Attempt to pair the keyboard again.
  • Run the Bluetooth Troubleshooter.
  • Run SDT or Windows Updates.
  • Replace the batteries or charge the keyboard.
  • Reset.

This ensures a consistent format for capturing and reviewing case information. Performing this standard procedure helps prevent instances where a step was neglected to be performed.

2. Check known issues.

From KB 4072657 Informational: Surface: Technical guides and known issues landing page, select the customer’s Surface model and check for the issue.

There may be a current known issue that the customer is experiencing, with different troubleshooting steps.

Check known issues regularly.

3. Verify the customer’s PC is compatible with the keyboard.

Most Microsoft Bluetooth Mice require a host PC that supports Bluetooth 4.0 connectivity.

Surface Pro 3 and newer support Bluetooth 4.0.

If the customer is using a non-Surface PC, check with the OEM to verify their make and model supports Bluetooth 4.0.

A Bluetooth 3.0 keyboard can connect to a Bluetooth 4.0 host. However, a Bluetooth 4.0 keyboard cannot connect to a Bluetooth 3.0 host.

As such, compatibility is important to a successful experience.

4. Restart the Host

Go to Windows > Power > Restart

Restarting the PC refreshes any drivers or FW that may be having issues.

5. Attempt to pair the keyboard again

If already paired, you may need to remove the Keyboard first:

1. Go to Settings > Devices > Bluetooth & Other Devices.

2. Select the keyboard.

3. Select remove.

4. When asked if you want to remove the keyboard, select Yes.

[Asset 4346174]

For information on how to pair in Windows 10, go here.

If the restart was successful, and refreshed the drivers, you may be able to successfully pair the keyboard to the host.

6. Run the Bluetooth Troubleshooter

1. Go to Settings > Update & Security > Troubleshoot.

2. In the list of available troubleshooters, select Bluetooth.

3. Follow the on-screen instructions.

4. The troubleshooter will tell you the result of the test. If resolved, attempt to pair the keyboard again. If no resolution, or is unable to resolve, go to the next step.

[Asset 4346175]

 

The Bluetooth Troubleshooter will run a series of scripts and actions that will attempt to locate and resolve a Bluetooth issue on the host PC.

The troubleshooter will also send Microsoft Bluetooth status logs and other data about Bluetooth on their device if the customer has opted into sharing data with Microsoft in Settings.

To opt out, customers can go to Settings > Privacy > Diagnostics & Feedback and configure their settings.

In addition, they can use Settings > Privacy to configure their privacy preferences.

If the customer does not want to run the app because of privacy concerns, do not force the customer into running the app.

7. Run SDT or Windows Updates.

For SDT:

See KB 4014334 Procedure: Surface: Using the Microsoft Surface Diagnostic Toolkit for information on SDT.

For Windows Updates:

1. Go to Settings > Update & Security > Windows Update > Check for Updates.

2. If updates are available, install them.

3. Make sure to restart the PC after installing updates if not prompted to do so.

Note: It’s a good idea to check for updates again or restart again after the first initial check and install. Sometimes updates require multiple restarts, or other updates to install first before installing any remaining updates.

Windows and Surface regularly ship updates intended to improve the customer experience.

There may be a Bluetooth update available that will resolve the customer’s issue.

Note: You are required to run SDT if supported in your region/ language. For more information, see KB 4014334.

8. Replace the batteries or charge the keyboard.

For mice with removable batteries:

Remove the batteries and replace with fresh ones. Most Microsoft mice use AA (Double A) batteries.

If the customer does not have AA (Double A) batteries or is unwilling to replace the batteries, follow steps below.

For mice without removable batteries:

Connect the keyboard to the host PC using the inbox cable. The keyboard may work while connected to USB. This does not resolve the issue. Let the keyboard charge for 15 – 30 minutes, and test pairing and Bluetooth behavior again. This may require a call back or follow up email.

Even though the low battery indicator is not present on the keyboard (for example, A Windows toast message, or a blinking amber LED), a keyboard with low power (for example, < 20% charge) may exhibit odd behavior.

9. Reset

Good: Fresh Start

Go to Start > Settings > Update & security > Recovery.

Under “More recovery options,” select Learn how to start fresh with a clean installation of Windows. Follow the on-screen instructions.

Better: Reset from within Windows

Go to Start > Settings > Update & security > Recovery.

Under Reset this PC, select Get Started. Follow the on-screen instructions.

Best: USB recovery

The customer will need to download and set up a recovery image on a USB drive. Send the USB recovery link to the customer and help them get started. Information on what size and type of drive is provided at the destination page.

-OR-

Set up an order to have a USB recovery drive sent to the customer.

Fresh Start (good):

  • Fresh Start is the least intrusive form of reset.
  • Customers will not lose any files or personal information.
  • Customers will need to reinstall desktop programs.
  • Fresh Start also ensures the customer has all available updates once complete.
  • If there is an issue with Windows itself, the issue may still occur after a reset.

Reset from within Windows (better):

  • This option uses the devices hard drive partition to reimage the device.
  • This option can delete personal data (The customer can choose Keep my files or Remove everything).
  • The customer will need to reinstall desktop apps.
  • The customer will need to run updates upon completion.
  • If there is an issue with the partition, the issue may still occur after the reset.

USB recovery (best):

  • This option reimages the entire device, including the partition on the hard drive.
  • It is the most complete form of reset.
  • Customers will need to back up their personal data.
  • Customers will need to reinstall desktop apps. Customers will need to run updates after completion.

If the issue occurs after a reset, then the root cause is very likely hardware related.

Is the issue resolved?

Yes, close the case: Once resolved, have the customer check for updates.

For SDT enabled sites, let the customer know about the Surface Diagnostic Toolkit. Fix common Surface problems using the Surface Diagnostic Toolkit.

For all other sites, check Settings > Update & Security > Windows Update for updates.

KB 4014334 Procedure: Surface: Using the Microsoft Surface Diagnostic Toolkit

Making sure Surface and Windows are up to date may help avoid this problem in the future.

No, replace the keyboard: Set up an exchange for the customer based on the status of their warranty.

See KB 4514977 Procedure: Surface: Services Exchange and Repair Offers Overview to create a service order.

There is likely a hardware problem with the Surface device.

Has the customer already replaced the keyboard and is having the same issue? Escalate the case: Work with a Tech Lead to make sure all necessary information is captured prior to escalation.

Capture the below information in your case notes when you escalate:

  1. Keyboard make and model
  2. Host PC make and model
  3. Bluetooth driver from Device Manager > Bluetooth.
  • On most Surface devices: this will be called “Marvell AVASTAR Bluetooth Radio Adapter"
  • On Surface Go, Surface Laptop 3 15 in. with AMD, and Surface Pro X: this will be called "Qualcomm Atheros QCA61x4 Bluetooth".
  • On Surface Laptop 3 15 in. with Intel, Surface Laptop 3 13.5 in., and Surface Pro 7: this will be called "Intel(R) Wireless Bluetooth(R)".
  • On non-Surface devices: this may vary. Look for something with “radio,” “radio adapter,” or “driver” in the title.
  1. Windows Version. Go to Settings > System > About and capture the full Windows Version information.

This information will be used to by the Tier 3 team to investigate the issue fully and work towards a solution.

Set proper expectations based on current Escalation processes.

8. Reset

Good: Fresh Start

Go to Start > Settings > Update & security > Recovery.

Under “More recovery options,” select Learn how to start fresh with a clean installation of Windows. Follow on screen instructions.

Better: Reset from within Windows

Go to Start > Settings > Update & security > Recovery.

Under Reset this PC, select Get Started. Follow on-screen instructions.

Best: USB recovery

The customer will need to download and set up a recovery image on a USB drive. Send the USB recovery link to the customer and help them get started. Information on what size and type of drive is provided at the destination page.

-OR-

Set up an order to have a USB recovery drive sent to the customer.

Fresh Start (good):

  • Fresh Start is the least intrusive form of reset.
  • Customers will not lose any files or personal information.
  • Customers will need to reinstall desktop programs.
  • Fresh Start also ensures the customer has all available updates once complete.
  • If there is an issue with Windows itself, the issue may still occur after a reset.

Reset from within Windows (better):

  • This option uses the devices hard drive partition to reimage the device.
  • This option can delete personal data (The customer can choose Keep my files or Remove everything).
  • The customer will need to reinstall desktop apps.
  • The customer will need to run updates upon completion.
  • If there is an issue with the partition, the issue may still occur after the reset.

USB recovery (best):

  • This option reimages the entire device, including the partition on the hard drive.
  • It is the most complete form of reset.
  • Customers will need to back up their personal data.
  • Customers will need to reinstall desktop apps. Customers will need to run updates after completion.

If the issue occurs after a reset, then the root cause is very likely hardware related.

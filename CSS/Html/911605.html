The default value of the cachedlogonscount registry entry has changed from 10 to 25 in Windows Server 2008

BUG #: 49129 (Content Maintenance)BUG #: 1154176 (Windows OS Bugs)
The default value of the cachedlogonscount registry entry has changed from 10 to 25 in Windows Server 2008. The minimum and the maximum range of the value remains the same. The cachedlogonscount entry is located under the following registry subkey:
HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\Current Version\Winlogon\
Note You can set the cachedlogonscount entry to be any value from 0 through 50.

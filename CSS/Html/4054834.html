1. Creating a new inbound Fed Ex shipping label

  1. Go to https://www.fedex.com/fedextrack/.
  2. Select Sign In in the upper right corner.
  • Access this document for username and passwords for the account:
  • Surface - KB 4054834 Shipping Labels appendix.docx
  • Xbox - KB 4054834 Shipping Labels appendix.docx

Note: Carefully enter the password. If the password is incorrectly entered three times all users will be locked out of the website.

  1. Select Shipping from the Top Menu, then select Create a Shipment.
  2. Fill out the From Portion of the form. Remember this is the customer’s information:
    • Country/Territory
    • Company name (if applicable)
    • Contact name
    • Address 1
    • Address 2 (if applicable)
    • Zip
    • City
    • State
    • Phone number
  3. Fill out the To portion of the form. Remember this is the service center’s information:
  • Country/Territory: United States
  • Contact ID: Select Microsoft Returns Center from the drop-down
  • Contact Name: Microsoft Returns Center/Flex El Paso
  • Address 1: 9640 Plaza Circle
  • Zip: 79927
  • City: El Paso
  • State: Texas
  • Phone: 9158581022
  1. Fill out the Package & Shipment Details:
    • Ship Day: Today’s date
    • Number of Packages: 1
    • Pricing Option: FedEx Standard Rate
    • Weight: 5lbs
    • Declared Value: Leave Blank
    • Service Type: FedEx 2Day
    • Package Type: Your Packaging
  2. Fill out the Billing Details:
    • Choose msreturns=394 from the Bill transportation to drop-down.
    • Enter the service request number related to the shipping label under Your Reference.
  3. Do not select anything from the optional items (Special Services, Pickup/Drop-Off, Shipment Notifications, and Rates & Transit Times).
  4. Complete your shipment.
    • Select Ship.
  5. Review and confirm the customer’s information is correct, then select Ship.
  6. Save the shipping label as an image locally using the customer’s name and the services request number as the file name.
  7. Email the shipping label to the customer.
  8. After emailing the shipping label to the customer, delete the image saved locally to your computer.

2. Creating a new inbound UPS shipping label

2.1 Creating a shipping label for a customer in Canada

  1. Go to UPS CampusShip.
    • Access this document for username and passwords for the account:
    • Surface - KB 4054834 Shipping Labels appendix.docx
    • Xbox - KB 4054834 Shipping Labels appendix.docx
  2. Select Create a Return.
  3. Select Enter New Address.
  4. Enter the customer’s physical address, email address, and phone number.
  5. Select Edit on Return to Address.
  6. Select the appropriate service center from the address book.
    • Microsoft Canada
    • Canada
    • C/O Telmar Tech/iQor
    • 99 Signet Dr
    • Suite 200
    • Toronto ON M9L1T6
  7. Select Update.
  8. In the What are you returning section, complete all the fields.
  9. Select UPS Express or UPS Standard for the Service Field.
  10. Select Electronic Return Label for the Label Delivery Method field.
  11. Use the service request number for both the RMA number and MS Store Order ID field.
  12. For the How would you like to Pay field, select Microsoft Canada.
  13. Select Next.
  14. Review all the above entered information for accuracy.
  15. Select Ship Now at the bottom. A UPS tracking number will populate on the next page.
    • If the label was created on behalf of another advocate, reply to their request with the tracking number.
    • After emailing the shipping label to the customer and confirming resolution, delete the shipping label from the computer.

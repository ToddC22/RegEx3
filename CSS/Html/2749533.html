What is this article about?

This article provides support boundaries for advocates who supports Microsoft account, outlines Microsoft account customer scenarios, and helps explain to a customer the benefits of having a Microsoft account. Use this as a reference guide to determine the scope of support for customers with questions about their Microsoft accounts.

3.1 Verify customer’s Microsoft account

1. Search for the customer's account in ASD.

2. If the customer's account is not found:

  1. Ask the customer if they use any other account names (aliases) to sign in and, if they do, let you know what they are so you can search for the real account name.

Note: ASD can only report primary alias status.

  1. Verify the spelling with the customer.
  2. Ask the customer if they have any other accounts (email address), phone numbers or gamertags so you can search for those.

3. Once the account is found verify it with the customer.

4. Assist the customer with updating security information to help avoid future security issues and improve future problem-solving efforts.

3.3 Account with no security proof (Light weight Microsoft account)

Verifying an account that does not have security information:

The user has a Microsoft domain Microsoft account (outlook.com or hotmail.com), and under Details in MST it says, "There is no security info available for this account". In this situation, the customer cannot recover the account if they have forgotten the password.

Educate the customer that they need to add security proofs via account.microsoft.com > Security > Security contact info.

How to close my account?

How to close your Microsoft account

The customer changed information on their Microsoft account (primary alias, name, address, or photo), and doesn’t see the changes on a service’s profile.

These changes are immediately shown on account.microsoft.com. Some services may take up to 24 hours to receive an updated token because they are configured for a once a day update.

Removing an existing alias on my Microsoft account.

 

Caution:

When removing an email alias that is a Microsoft domain (@Outlook.com, @Live.com or @Hotmail.com), that email is completely deleted from our systems and can no longer be used to send or receive emails. It will not be available for use again as an alias on another account. Microsoft does not recycle email addresses so it may not be reclaimed at a future time.

 

How to move my alias to another account to merge or consolidate accounts.

 
  1. Aliases cannot be moved from one Microsoft account to another. 
  2. If you remove your alias from the account it’s on today, it will be deleted from our system and is not recoverable. You will not be able to use it to send or receive emails.
  3. Microsoft does not recycle aliases once deleted. It will not become available at any time in the future.

4.2 What is the difference between a Microsoft account and a local account

Microsoft account

Local account

You’re in control of your Microsoft account - control of your account’s privacy settings and managing personal information. Your Microsoft account comes with state-of-the-art adaptive security with identity-theft protocols to help keep your information safer.

A local account is any username and password combination used to log into a Windows operating system. It grants you access to the system's resources on a specific PC and allows the user to customize settings and preferences. A local account does not enhance user security; in fact, this type of account allows a user to remove password protection if they don't want it.

All your devices and services work with Microsoft account, providing complete integration of all services into a single account.

Local accounts can be created for a single operating system only. A local account allows the user to install Desktop apps, personalize settings and use the operating system on that device.

If a user has multiple devices, they need to create a different local account for each one of them and settings will not be synchronized across the devices.

Use an email address instead of a username to log into the operating system, a user can choose to use either a Microsoft email address (hotmail.com, outlook.com), third-party like yahoo.com or an ISP specific email address like Verizon.com to create their Microsoft account.

Email account not required.

A local account is generally tied to a username that is specific to that computer such as Admin.

Microsoft accounts allow a two-step verification system of your identity each time you sign in.

Two step verification is not available with a local account.

Microsoft accounts provide the ability to download, install, and restore modern apps from the Windows Store.

A user can access the Windows Store but cannot download and install apps with a local account.

Microsoft accounts provide access to Cortana on Windows 10 devices.

Access to Cortana is not possible with a local account.

Microsoft accounts enable a user to setup and manage their family's digital life across Windows PCs, Xbox, and mobile devices, facilitates children’s permissions for websites, games, apps, and movies.

The ability to manage a family’s digital life or oversee children’s activity and security is not available with a local account.

Scenario: Windows Local account recovery (also known as Offline Account).

When the customer is having trouble with local account recovery, use the following public content Change or reset your Windows password.

A customer may need to create a new account if they are unable to successfully reset their password with the steps provided. Setting up a new account on Windows using a Microsoft account gives the customer additional recovery options as well as all the features built into a Microsoft account.

4.3 Microsoft account FAQ

Why should I create a Microsoft account?

A Microsoft account gives you access to all your Microsoft services and subscriptions while tying all your settings together across all your Microsoft devices. It also allows you to manage your Microsoft purchases in one place.

http://microsoft.com/account

How do I create a Microsoft account?

Customers can either use an existing email account such as google.com yahoo.com as their Microsoft account name or create a new outlook.com email address.

If a customer already has a live.com, hotmail.com, or outlook.com email address, they already have a Microsoft account.

You can direct customers to the How to sign up page if they seem unsure or offer to walk them through the sign-up process starting at https://signup.live.com.

Where do I manage my Microsoft account?

https://account.microsoft.com/

How do I change my account's country/region?

Ask the customer to review the country/region help content for Xbox at How to update your Microsoft account if you’re moving to a new country or region, which explains possible issues to be aware of when changing an account's country/region.

To change the country/region settings on your Windows device, go to Settings>Time & language> region & language.

This does not change your billing account region settings.

It is recommended that customers use the Xbox country/region change process even if they don't have an Xbox.

This method also changes billing info, which may not be changed using other methods.

Xbox Country Change Form

How do I change my email address or add an alias to my account?

Change the email address or phone number for your Microsoft account.

Can I make an additional account?

One account should be all you need, if you want a new email address, you can add aliases to your account. For more information on aliases visit Change the email address or phone number for your Microsoft account.

If you want to create an account for a family member or a child and link that account to your account, follow the instructions in Set content restrictions on Windows 10 and Xbox One.

Add a trusted device to your Microsoft account

4.5 Hard messaging after unsuccessful troubleshooting

You may have to deliver the difficult news that it might be time to create a new account if the recovery flow did not resolve the customer’s issue. By demonstrating empathy and educating customers, it may help reduce the customer's frustration when receiving this message.

Summarize all the steps taken with the customer to try to recover their account. See KB 3107652 Procedure: Microsoft account: Customer cannot regain access to account.

When creating a new account with the customer, emphasize the importance of creating a strong password and always having updated security information (see section 5 for more information) in order to avoid encountering these potential issues in the future.

5.2 Password creation guidance

Instead of recommending that customers change their passwords frequently, teach them to create strong passwords that they can remember. Frequent changes lead to weak or forgotten passwords.

  • Length matters. Choose a password that is a minimum of eight characters long.
  • Choose your password freely, it is unique and one you haven't used before.
  • Use a mix of words, special characters, uppercase and lowercase letters, and numbers. (But don't use them in a sequence. For example, 12345 or 012012).
  • Avoid using common terms such as your name, birth date and company name. Try to make it as unique as possible.
  • Consider using multiple words or a phrase (for example, GreenElephantSummer23).

How to prevent Groove from starting automatically when you start Windows

This article describes how to prevent Microsoft Office Groove 2007 and Groove Virtual Office from starting automatically when you start Microsoft Windows.
To prevent Groove from starting automatically when you start Windows, follow these steps:
  1. On the Options menu, click
    Preferences, and then click the Optionstab.
  2. Click to clear the Launch Groove when Windows starts up check box.

1. What is Microsoft Teams

Microsoft Teams is a communication and collaboration app with features such as chat, video calling, file sharing, storage, and more. It expands upon the Microsoft 365 services by including Teams features for your personal life, allowing users to stay organized and connected with family and friends.

It is available for mobile on Android and iOS and desktop on Windows, macOS, and web. If users have a work account for Teams, they can use Microsoft Teams to switch between using their personal Microsoft account and their work account in the same app.

Users can sign up and download Teams here.

2. How to use Microsoft Teams (mobile)

2.1 Signing up for a Microsoft Teams account

Personal account

Upon downloading and installing the app and launching it for the first time, it will detect if there is an AAD (Azure Active Directory which is a work account) or Microsoft account on the device that can be used to sign in.

  1. If no account is detected, customers can select Sign up for free at the bottom to create a new account.

[Asset 4561359]

  1. You can create a Personal or Business account. Select Personal for this flow.

[Asset 4561369]

[Asset 4561495]

[Asset 4561496]

If the user has been using their AAD on the device (such as with Outlook or other apps), it should show up for them to select. If not, they can select Sign in with another account and go through a standard AAD authentication workflow.

Switching between a Microsoft account and Azure Active Directory with Microsoft Teams

If the user is using Teams for both personal and work, they can easily switch between a Microsoft account and AAD using Teams.

  1. In Teams, select the hamburger icon on the upper left.

  1. Their profile will appear under MY ORGS. To switch between the Microsoft account or AAD select the profile (pointed out with black arrows).

4. Hardware requirements for Microsoft Teams

Microsoft Teams is initially available for mobile on Android and iOS but will be available later for online and desktop. For the best experience with Teams, use the latest version of iOS and Android.

  • Android: Compatible with Android phones and tablets. Support is limited to the last four major versions of Android. When a new major version of Android is released, the new version and the previous three versions are officially supported.
  • iOS: Compatible with iPhone, iPad, and iPod touch. Support is limited to the two most recent major versions of iOS. When a new major version of iOS is released, the new version of iOS and the previous version are officially supported.
  • Desktop/web
    • Desktop client: Windows 10 version 1803 and later, macOS Sierra 10.12 and later, Linux is web only.
    • Web client: Any modern browser (IE is not supported).

 

See the following link for more information on hardware requirements: Hardware requirements for Microsoft Teams.

5. Troubleshooting account login issues with Microsoft Teams

Since accounts in Microsoft Teams are Microsoft accounts, troubleshooting account sign in with Microsoft Teams is the same as troubleshooting login with Microsoft account across all Microsoft services.

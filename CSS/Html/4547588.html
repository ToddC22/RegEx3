1. What is the Windows 10 May 2020 update (version 2004)?

The Windows 10 May 2020 update (version 2004) is the latest update Microsoft is releasing for Windows 10 devices. Notable updates include Restartable apps, Cortana as app, Pairing Bluetooth Devices, Virtual Desktops, Managing Notifications, Network Status Interface change, and an update to the Xbox Game Bar.

Customers on Windows 10 will receive a notification from Windows Update when the update becomes available. Customers can then follow the prompts to update their device.

Customers can preview new builds by joining the Windows Insider Program. See https://insider.windows.com.

2.1 Restartable apps

There is a new setting that allows users to have Windows automatically restart Windows 10 apps when signing back into Windows.

To enable this feature:

  1. Go to Settings, then Accounts.
  2. Select Sign-in options.
  3. Under Restart apps turn on Automatically save my restartable apps when I sign out and restart them after I sign in.

[Asset 4559336]

  1. Start one or more Windows 10 apps, such as Feedback Hub.
  2. Sign out and then sign back into Windows.

Window 10 apps that were started, such as Feedback Hub, should restart minimized with a taskbar button.

2.2 Cortana as app

Cortana is decoupled from the operating system and is now a standalone app.

  • The app does not have all the same features as before, but new features will continue to be added in the future.
  • Cortana now supports both light and dark themes in Windows.
  • Cortana has new speech and language models, and significantly improved performance, making it faster and more reliable than before.
  • The Cortana app can be moved and resized just like you can with other open windows.
  • Language/region support is only English (US) for now, but other language/regions will come in the future.

2.3 Pairing Bluetooth devices

Improvements have been made to the Bluetooth pairing process to make it faster and more helpful for supported devices.

  • Pairing is all done in notifications now. You don’t need to go to Settings app anymore to finish pairing.
  • For faster pairing time, we improved the UI by showing one less notification.
  • We added a Dismiss button to the first notification to give you more control while using Swift Pair.
  • To make the notification more helpful, now we show the device name and category when we can.

Supported devices include:

  • Surface Ergonomic Keyboard
  • Surface Precision Mouse
  • Surface Mobile Mouse
  • Surface Arc Mouse
  • Surface Headphones
  • Microsoft Arc Mouse
  • Microsoft Modern Mobile Mouse
  • Microsoft Bluetooth Keyboard
  • Microsoft Bluetooth Mouse

2.4 Virtual Desktop

You can now rename Virtual Desktops:

  1. Open Task View by selecting Task View in the taskbar or pressing WIN+Tab.
  2. Select New Desktop.
  3. Select the desktop name (“Desktop 1”) and it should become editable, or right-click the desktop thumbnail and a context menu will appear with a Rename entry.
  4. Input whatever name you’d like, and press enter.

2.7 Xbox Game Bar

There is a new Xbox overlay for gaming that includes an FPS counter and achievements. To enable this, you need to do the following:

  • Make sure you’re on Windows 10 Version 2004 (Build 18990 and above).
  • Set focus to the game you want the overlay on and press WIN + G.
  • To get Game Bar updates, you may need to join Insider Content in the Xbox Insider Hub app from Microsoft Store.

Note: Xbox advocates will provide full support for the Xbox on Windows 10 experience. While Windows TS provides support for Xbox on Windows 10 experience using their best effort, Windows TS advocates should transfer to Xbox support for collaboration when they cannot resolve the customer’s case. Before the transfer, Windows TS should make sure the user is on build 18990 or above.

2.8 Reset PC from the cloud

When using the Reset this PC option, customers can now choose to download Windows over the internet or use their local reinstall.

[Asset 4559341]

  • The cloud download option will reinstall the same build, version, and edition, that is currently installed on your device.

Note: This is a recovery operation and will remove apps you have installed. Additionally, if the “Remove everything” option is selected, user data will also be deleted.

2.9 Connecting network cameras

Customers can now associate network cameras to their PC, enabling photo capture and streaming of video in camera applications. Currently Windows only supports ONVIF Profile S compliant cameras, which are standards-compliant network cameras optimized for real-time streaming video capture.

To connect a network camera without authentication, perform the following:

  1. Open Windows Settings and choose Devices.
  2. Inside Bluetooth & other devices → Everything else, select Add device to discover network cameras connected to your LAN.
  3. Select the network camera you want to pair, and your device should now be connected.

Note: If you are not able to help customers solve their issue with basic troubleshooting, provide the customer with this email for them to contact directly for additional support: onvifsupport@microsoft.com.

2.10 Task Manager

Task Manager has a couple of new features.

Temperature monitoring support for GPU

[Asset 4559342]

  • Showing the GPU temperature in Task Manager is currently only supported if you have a dedicated GPU card.
  • You may need an updated graphics driver to see the temperature, specifically one which supports version 2.4 (or higher) of WDDM is required. To check which version your PC has, press WIN+R and enter dxdiag. Under the Display tab, it will say your WDDM version next to Driver Model.
  • Currently the temperature value is only supported in Celsius.

Disk types (for example, SSD) within Task Manager

You can now see the disk type listed in Task Manager. In the example below, you see that disk C is identified as an SSD. This is helpful if you have multiple disk types, and you want to differentiate the disk types.

[Asset 4559343]

Support for new AL/ML hardware chipsets

This is a new class of hardware and will be classified differently from GPUs in the Performance tab of Task Manager. This mainly affects high end devices.

2.12 Keyboard hotkey for collecting logs

There is a new keyboard hotkey to collect logs, WIN + Ctrl + Shift + L. With the hotkey, the customer can start a trace, stop the trace, and then share the reference number with a support advocate. Only supported for local sessions that have connected to the network and power, and for whom Diagnostic Data is set to Full.

Note: TS Support advocates cannot access the logs, and it can only be accessed by the engineering teams so this feature should be limited to be used only by Tier 3 support since it will not help Tier 1/Tier 2 to troubleshoot the customer’s issue.

2.15 Sandbox

The following was added to improve Windows Sandbox Accessibility scenarios:

  • Microphone support​: You can configure the audio input device via the Windows Sandbox config file.
  • Shift + Alt + PrintScreen key sequence: Activates the ease of access dialog for enabling high contrast mode in Windows Sandbox​.
  • Ctrl + Alt + Break key sequence: Allows entering/exiting full screen mode in Windows Sandbox.

Windows Sandbox configuration file support:

​Windows Sandbox now supports configuration files which allow customers to configure some aspects of the sandbox, such as vGPU (Virtual GPU), Networking, Shared Folders, and Startup Script. Additional information can be found here: https://techcommunity.microsoft.com/t5/windows-kernel-internals/windows-sandbox-config-files/ba-p/354902.

2.16 Input and language setting

There are changes made to various languages settings.

Language settings user interface

The new overview section lets you quickly know which languages are selected as default for their Windows display, Apps & websites, Regional format, Keyboard, and Speech. If everything looks as expected you can quickly move on, you can select one of the tiles which will take you to where you need to be to change that selection.

[Asset 4559349]

Reorganized the language features installation page a bit and added tool tips with descriptions.

[Asset 4559350]

Japanese IME Improvements

  • Prediction Candidate window key navigation: We fixed an issue where the focus inside prediction candidate window wouldn’t move when using the up arrow key.
  • Key customization: We’ve improved the discoverability of key assignment settings. Also, we’ve updated the default assigned value of Ctrl + Space to be None. Ctrl + Space can still be used for toggling IME-on/off by changing the value through its setting.

[Asset 4559351]

Chinese IME Improvements

  • Bopomofo, ChangJie, and Quick IMEs: We improved the speed of toggling conversion mode ([中] mode and [英] mode). Now it you can switch between the two faster.
  • Bopomofo IME: Added a new setting to disable the keyboard shortcut (Note: The default is enabled). To access this setting, right click the [中/英] icon in the taskbar, select Settings > General > Use Shift + Space to switch character width.
  • Added the ability to change the candidate font size, separate from the rest of the system if you find that you need it to be a bit bigger. (You can still change the system font size under Ease of Access > Display > Make text bigger.)

2.18 Passwordless Sign-in with Microsoft account

For improved security and a more seamless sign-in experience, you can now enable passwordless sign-in for Microsoft accounts on your Windows 10 device by going to Settings > Accounts > Sign-in options, and selecting On under Make your device passwordless.

[Asset 4559355]

This will strengthen your device sign-in by switching all Microsoft accounts on your device to modern multifactor authentication with Windows Hello Face, Fingerprint, or PIN, and eliminating passwords from Windows.

3. Who can get this update?

All existing Windows 10 devices running Windows 10 Home, Pro, and S users who have installed Windows 10 can get this update for free. Customers will receive a notification from Windows Update when the update becomes available. If the customer wants to update before the notification, they can use the Windows Update Assistant to get the update when it becomes available.

4. Getting the Windows 10 May 2020 update (version 2004)

Customers will receive a notification from Windows Update when the update becomes available. Customers can follow the prompts to update their device.

  • Reminder: You can always check what version the customer has by Start > Settings > System > About [look in the Version field].

If the customer doesn’t want to wait for it to be offered through Windows Update, they can:

  1. Look for it manually with Windows Update.
  2. If they don’t find it in Windows Update, they can update using the Windows Software Download site.

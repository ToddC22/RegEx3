How to uninstall SharePoint Workspace 2010

BUG #: 127389 (Content Maintenance)
This article describes how to uninstall Microsoft SharePoint Workspace 2010. Additionally, this article describes how to leave or remove all associated data when you uninstall SharePoint Workspace 2010.
In certain scenarios, you may have to uninstall SharePoint Workspace 2010.By default, when you uninstall SharePoint Workspace, all account data and Groove workspace data is left on the computer. This data can be used by a later installation of SharePoint Workspace. However, before you uninstall SharePoint Workspace we recommend that you safeguard the data by taking one of the following actions:
  • Save your account to a file.
  • Save any Groove workspaces which have no other members to an archive.
For complete procedures, see the following topics in SharePoint Workspace Help:
  • Saving a backup copy of your account
  • Saving a workspace as an archive
After you back up any data that you may want to save, use one of the following procedures to uninstall SharePoint Workspace from the computer.

How to uninstall SharePoint Workspace but leave all associated data

  1. On the File tab, click Exit.

    NoteIf SharePoint Workspace stopped working because of an error, open Task Manager to confirm that the Groove.exe process is not running.
  2. In Control Panel, open the Programs and Features item or open the Add or Remove Programs item.
  3. In the list of installed programs, select the Microsoft Office 2010 suite that you are running, and then click Change.

    Note If you installed a stand-alone version of SharePoint Workspace, click SharePoint Workspace 2010, click Uninstall, and then go to step 5.
  4. Click Add or Remove Features, and then click Continue.
  5. In the Microsoft Office SharePoint Workspace list, click Not Available, and then click Continue.
  6. When the Setup program is finished uninstalling SharePoint Workspace, click Close.

How to uninstall SharePoint Workspace and remove all associated data

  1. On the Groove Launchbar, click the File tab, click Options, and then click Preferences.
  2. On the Identities tab, click No Listing for The Public Workspace Contact Directory option and for The local network directory option.

    Note If you have multiple identities from a previous installation of Groove, repeat this step for each identity.
  3. On the Account tab, click Delete under Delete Account from this Computer.

    Note If you delete your account, your current installation will not be retained.
  4. Click OK to exit Preferences, and then click OK to exit the SharePoint Workspace Options dialog box.
  5. On the File tab, click Exit.
  6. Restart SharePoint Workspace to complete the communications associated with the account deletion.
  7. On the File tab, click Exit.
  8. In Control Panel, open the Programs and Features item or open the Add or Remove Programs item.
  9. In the list of installed programs, select the Microsoft Office 2010 suite that you are running, and then click Change.

    Note If you installed a stand-alone version of SharePoint Workspace, click SharePoint Workspace 2010, click Uninstall, and then go to step 11.
  10. Click Add or Remove Features, and then click Continue.
  11. In the Microsoft Office SharePoint Workspace list, click Not Available, and then click Continue.
  12. When the Setup program is finished uninstalling SharePoint Workspace, click Close.
  13. To remove all data files, delete your Groove data folders. These folders are in one of the following locations, depending on the operating system that you are running:

    Windows Vista and Windows 7
    C:\Users\username\AppData\Local\Microsoft\Office\Groove
    Windows XP
    C:\Documents and Settings\username\Local Settings\Application Data\Microsoft\Office\groove

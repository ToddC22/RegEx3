3.11 Windows Sandbox

Windows Sandbox is a new lightweight desktop environment tailored for safely running applications in isolation. It's an isolated, temporary, desktop environment where you can run untrusted software without the fear of lasting impact to your PC. Any software installed in Windows Sandbox stays only in the sandbox and cannot affect your host. Once Windows Sandbox is closed, all the software with all its files and state are permanently deleted.

4.4 Third-party apps

Excluding games from the Windows Store App.

Important Note

When discussing third-party apps, do not present your personal opinion on third-party apps or say anything about third-party apps that can be viewed as negative. Do not make claims that Microsoft products are better than third-party products or make any comparisons that may underrate third-party apps

  • For app-related blocking issues when installing/upgrading Windows:

Before uninstalling, make sure that the customer has program installation media and other materials needed to reinstall, should that be needed later. This might include installation DVDs, Product Key(s) and so on.

  • Use this scripting before disabling/uninstalling:

“As part of troubleshooting the issue, we will temporarily disable/uninstall [insert third-party app/software name]. We just want to make sure we create an environment where we know the updates should be installed.”

  • Use this scripting if the issue is resolved after disabling/uninstalling:

“It appears the updates started installing after disabling/uninstalling [insert third-party app/software name]. While I cannot definitively state [insert third-party app/software name] caused the updates from not installing, the installation worked after temporarily disabling/uninstalling. I suggest you contact [insert third-party app/software publisher name] and explain the issue you faced and how temporarily disabling/removing allowed to have updates installed.”

  • Refer to the publisher of the app for issues and questions with features and functionality inside the app.
  • If you have exhausted standard/expected troubleshooting steps, refer the customer to the publisher of the app.
  • For third-party games offered via the Store app, please refer to the following support boundaries see KB 3055957 Procedure: Original Xbox Support Boundaries support boundaries.

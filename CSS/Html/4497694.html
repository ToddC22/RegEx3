Procedure: Advise Me: Stay Current Keep Windows 10 Up-to-date Campaign

[Asset 4049349]

Who should use this article?

  • Accounts & Billing
  • Microsoft Store
  • Sales
  • Windows

What is this article about?

This is an Advise Me campaign designed to educate customers running Windows 10 to check for updates to ensure they’re running on the latest version of Windows 10. This only applies to Windows 10 customers who aren’t on the latest version.

[Asset 4133248]

1. Talking Points: Stay Current Keep Windows 10 up to date

1.1 Create your own pitch using the features and benefits below:

  • The latest Windows 10 updates include anti-virus, firewall, Windows Defender, and anti-phishing technologies – all delivered built-in at no extra cost to you.

 

  • The latest updates are available for free since you’re already running Windows 10.

Customer ready pitch example:

“I noticed you're not running the latest version of Windows 10. It’s free to update and you’ll stay protected. You can check for the latest Windows 10 update here, https://aka.ms/Windows10StayCurrent.”

[Asset 4133250]

Return to top

1.2 Is the customer interested?

Unless the customer explicitly says they are not interested, provide the following information:

  • “You can check for the latest Windows 10 update here, https://aka.ms/Windows10StayCurrent.”
  • “I’ll also send you the link in the case email, so you have it handy.”

If the customer is not interested, include the links in your support email in case they want to learn about the feature later.

2. Email content (required)

Add this content to your case closure email.

Check for the latest version of Windows 10 here.
“Following up on our conversation today, here’s how you can check for the current version of Windows 10 at no cost to you. By staying up to date, you get anti-virus, firewall, Windows Defender, and anti-phishing technologies. Check for the latest Windows 10 updates.”

3. ASD Coding

Under Reason for Contact in the ASD form:

  1. Under Reason for Contact > Achieve More Conversation > Yes
  2. Achieve More Product > Windows
  3. Achieve More Campaign > Stay Current
  4. Customer Interested > select Yes or No based on the customer response

Details on where to code an Advise Me (Achieve More) conversation in ASD can be found in KB 3082289 Informational: Advise Me Educate Program, section 6.

Microsoft Teams is a hub for teamwork; it’s an app that helps users pull together a team and work together using chat instead of email and channels instead of file folders. It’s a workspace where team members can securely edit files at the same time and see “likes” “@mentions”, and “replies” with a single tap. It’s a place where users can customize the experience by adding notes, websites, and apps. TEAMS is cloud-based to allow everyone to work together.

For more information see: Microsoft Teams Introductory Training Plan

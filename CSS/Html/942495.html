Windows Mail and Windows Live Mail error codes

BUG #: 82167 (Content Maintenance)
This article contains a list of error codes that may occur in Windows Mail or in Windows Live Mail. The list contains the following categories:
  • Error code
  • Error type
  • Description
The following list of error codes may be returned in Windows Mail or in Windows Live Mail if a failure occurs when you send mail or when you receive mail.

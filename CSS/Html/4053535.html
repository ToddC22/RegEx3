2. Using the Lookup tool 

After collecting the 5x5 token from the customer, enter it in the token text box and select 5x5. If the token is valid, results will appear providing the status of the token.

[Asset 4132600]
If the token is not valid, messaging will appear stating: The provided token information is invalid, token not found. 

Return to top

3. Deactivate token

To deactivate a token, look up the token and select Deactivate.

Verify the correct token number is showing on the screen, enter a reason for deactivation in the Comment section.  Select Submit to complete token deactivation.

Return to top

4. Issue token

To issue a new token follow the steps below:

  1. Go to Services: Xbox Live, click on the action menu [Asset 4494316] and select Issue Token
    [Asset 4494317]
  2. Choose the Token Type as Content Token or Subscription Token. [Asset 4494333]
        [Asset 4494332]
  3. Select the relevant content type from the Content Options dropdown list:
    • Xbox/Windows games
    • Xbox Game Pass
    • Xbox Live Gold
    • EEA Play (formerly EA Access)
      [Asset 4494321]                                                                                                                                            [Asset 4494323]     
  4. Select a Reason from the dropdown list (See Reason policy in this article, section 5). [Asset 4494324]
  5. Write a justification Comment and include the SR number.
  6. Select Next.
  7. Make sure all of the information listed and if correct, select Submit (if you need to edit the details just click Previous).
    [Asset 4494213]
  8. You will receive a message stating if the token was successfully applied or not.

Note: If you get a Country Mismatch error message, refer to the following articles:

  • KB 4039267 Change your country or region in Microsoft Store
  • KB 4027266 How to change your country or region for Microsoft Store
  • KB 3072567 Procedure: Xbox: Engagement scenarios

5. Reason policy

  • Promotional token expired: See section 3.4 of KB 818928 Troubleshooting: Prepaid, trial, in-game, and subscription redemption issues.

[Asset 4494326]

  • Did not receive promotional token: This is most used at ACA level for Partner Requests. For example, the Dell Forza Horizon promotion. See KB 3151186 Troubleshooting Xbox One bundle content issues.
  • Damaged token: See section 3.3 of KB 818928 Troubleshooting: Prepaid, trial, in-game, and subscription redemption issues.
  • Missing token from bundle: See KB 4036047 Process: Handling Dynamic Store Bundling (DSB) redemption issues and 3151186 Troubleshooting Xbox One bundle content issues.
  • Replacing subscription time: This can be used:
    • If the system cancelled/locked the subscription to migrate from Legacy to Modern platform
    • When doing a Region Migration
    • The customer was charged but the transaction status is not Purchased
    • For another applicable MIR that might apply
  • Entitlement issue: See KB 921272 Troubleshooting: Xbox downloadable content (DLC) license/entitlement issues.
  • Advanced Capability Advocate Make Good: MIR at ACA discrepancy. This is mostly used in account take over cases.

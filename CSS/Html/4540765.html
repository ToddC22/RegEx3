2. Definitions

  • Microsoft Merchant Center – Allows the advertiser to create a catalog, which includes images and other information about their products.
  • Product Partition Report – It shows the impressions, clicks, spend, average cost-per-click and conversion for each product group in your Shopping Campaigns.
  • Benchmark Bids – To see the performance data of the product groups in your shopping campaigns.
  • Untargeted Offers – A downloadable diagnostic report that shows the number of offers in a store that is not targeted in your existing or new campaigns. This report will be generated once in 24 hours.
  • Catalog Rejections – These depend on the target location/countries where the catalog will be used for. These error messages can be either about “Adult Image”, “Weapon Image” or “Adult Content”.
  • Feed Rejections – These are error or warning messages when trying to upload feed file.
  • Merchant Promotions – You can promote products directly from your Microsoft Shopping Campaigns’ inventory with special offer tags. Merchant Promotions are free to add to your product ads, with the usual charges for any clicks you get. Including Merchant Promotions can improve the visibility of your ads, which can lead to more clicks and improve your return on investment (ROI).

4. Running a shopping audit report

4.1 Gathering of requirements

Here you will find instructions on how you can gather all requirements necessary before conducting a shopping audit report.

Account managers, specialists or frontline advocates will submit a request to the BOS team. Once received, the following steps should be completed.

Review of submission form

Review the submission form. The requester will specify the details of the account to be used for the shopping audit report, along with other instructions.

  • Check if all required fields were completed.
  • If required information is missing, reach out to the requester.

Pulling up the account

Access the account through Microsoft Advertising UI.

4.2 Running a Product partition report

  1. From the UI, search for the account and navigate your way to the Reports tab. Download a Production partition report by expanding the Product ads category and selecting Product partition as the report type.

[Asset 4541318]

  1. Under General settings, run a product partition report using the settings below:
  • Unit of time: Summary
  • Date range: Last 30 days (unless otherwise specified in your ticket)
  • Time zone: Leave as is
  • Format: Excel
  • What to report on: Specific accounts and campaigns

Note: For Date range, you may run the report using Last 14 days in case the Last 30 days resulted to nothing.

[Asset 4541319]

  1. Be sure to add the following columns from the Performance Statistics list of columns:

  • Benchmark bid
  • Impression share
  • Impression share lost to budget
  • Impression share lost to rank
  • Return on ad spend

[Asset 4541320]

  1. Before the file is downloaded, sort all 20 columns in the following order:
  • Account
  • Campaign
  • Ad group
  • Product group ID
  • Product group
  • Product partition type
  • Current Max CPC
  • Benchmark bid
  • Impression share
  • Impression share lost to rank
  • Impression share lost to budget
  • Impressions
  • Clicks
  • CTR
  • Avg. CPC
  • Spend
  • Conversions
  • Conversion rate
  • CPA
  • Return on ad spend

[Asset 4541321]

  1. Download the report and rename the tab as Benchmark Bids. This excel file will serve as the actual base of your final deliverable.

4.4 Generating Untargeted Offers report

  1. From the Reports tab, navigate your way to Tools then select Microsoft Merchant Center from the dropdown menu.

[Asset 4541327]

  1. Once you’re in the Microsoft Merchant Center page, select the Store Name which will open the selected store’s page for you.

[Asset 4541346]

Note: There will be instances where an account would have multiple stores. The steps indicated in this article must be done across all stores, from generating untargeted offers down to downloading merchant promotions. This information must be collated and saved in one file. If the line items exceed Excel’s limit, then it must be called out to the requester that the file(s) was saved in very large file share (VLFS). Make sure that the requester has access to VLFS before sending your closing email.

  1. Still inside the same Microsoft Merchant Center store, go to the Store summary M2 testing tab and select Catalog Summary V1. This is where you will find the list of catalogs in a store.

[Asset 4541357]

  1. Once you see a View errors link in the Status column, select the actual Catalog name to view both Catalog and Feed rejections.

[Asset 4541358]

  1. To get the Catalog rejections, select Download Details beside Download Summary.

[Asset 4541365]

  1. Open the file and move the tab to your final deliverable and rename the tab as Catalog Errors.
  2. Filter and sort the Type column from A to Z so that Error appears on top.

[Asset 4541366]

  1. To get the Feed rejections, select Download Details under the Feed file details field and follow steps 2 to 3. Rename this tab as Feed Errors.

[Asset 4541367]

Note: In cases where there are multiple catalogs that have a View errors link in a single store, make sure to download every available Catalog and Feed rejections. Collate them in one tab (one for catalog and the other for feed rejections), following the steps below:

  1. Delete the first three rows. Then, copy the details under the three new columns up to the last row.

4.6 Merchant Promotions

  1. From the same Microsoft Merchant Center store, select the Promotions tab.

[Asset 4541373]

  1. Verify if the advertiser has any promotions set up. If they do not, state this in the Merchant Promos tab of the final report as No report generated. See sample screenshot in step 3 of Generating Untargeted Offers report.
  2. If promotions are set up, export all of them by selecting the top box. Once everything is selected, select the Export button.

[Asset 4541374]

  1. A file in .tsv will start downloading and will open in Notepad. To convert this into an Excel file, go to the File menu and select Save As. A new window will open, and in the bottom right, update the Encoding dropdown menu from ANSI to Unicode. Add .csv in the file name and select Save.

[Asset 4541375]

[Asset 4541376]

  1. Open the file back in Microsoft Excel and move the Promotions tab to the final deliverable. Rename the tab to Merchant Promos. Using styles, highlight in green any promos with an Approved state. On the other hand, set those that are In review, Pending, Paused or Rejected in red.

[Asset 4541377]

  1. Highlight in red any promos with an end date within the next 30 days from the date the report is being worked on.

For example, the date of the report is October 21st, 2019. However, the next 30 days would include dates from October 22 nd, to November 21 st, 2019.

[Asset 4541378]

  1. Lastly, highlight in red any promos without any matching products found in the Products Matched Status column.

[Asset 4541379]
